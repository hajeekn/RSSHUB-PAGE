<div class="topicContent front_content"><p><span style="background-color: initial; letter-spacing: 1.3px;">　　Xbox Game Pass 服务一直广受玩家好评，只要订阅该服务，即可畅游百余款游戏，其中不乏《忍者龙剑传》系列、《神鬼寓言》系列等经典游戏。</span><br></p><p>　　近日，Xbox 负责人菲尔·斯宾塞在采访中聊了聊自己对 Game Pass 的想法，认为这一服务可以帮助他们考虑重启一些经典游戏。</p><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715100546456_u93176.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><blockquote><p><span style="background-color: initial; letter-spacing: 1.3px;">　　看到更多人在玩《掠食》《耻辱》《神鬼寓言》或更多那样的游戏，我们作为一支富有创造力造力的团队，能看到玩家对什么游戏感兴趣。<br></span>　　Game Pass 服务为我们提供了更多的数据，来让我们考虑那些或许能重拾起来的经典游戏，并结合新的创意和有想法的团队去着手推进。</p></blockquote><div class="vg_insert_img"><figure contenteditable="false"><img style="width:600px;" src="https://img01.vgtime.com/game/cover/2021/07/15/210715100515215_u93176.jpg" class referrerpolicy="no-referrer"><figcaption contenteditable="true" class="vg_insert_img_notice"></figcaption></figure></div><p>        IP 重启在 Xbox 中并不少见，比如《微软模拟飞行》《忍者蛙》以及最近公布的《完美黑暗》。 无论是单独的作品还是像《Rare 游戏合集》这样的经典作品合集，Game Pass 服务未来将可能拥有更多丰富多样的作品。  <br></p><blockquote><p>　　《Rare 游戏合集》对我们来说是一件非常有趣的存在，它可以让人们去体验 Rare 工作室的一些经典 IP 。我们会坚持下去，因为 Game Pass 为我们提供了一个有意义的商业模式。

</p></blockquote><p style="text-align: right;">来源：<a href="https://www.gamespot.com/articles/phil-spencer-says-seeing-older-games-succeed-on-game-pass-could-help-greenlight-reboots/1100-6493916/">Gamespot</a></p></div>
