<p>微软宣布，中国台湾Xbox Game Pass及Xbox Live Gold服务预计从2月起开始下调。整体调降幅度约1/3，例如主力的Xbox Game Pass Ultimate价格从原本的1个月新台币459元（约合人民币106元）降低至新台币299元（约合人民币69元），Xbox Game Pass（PC/主机）从原本的1个月新台币320元（约合人民币74元）降低至新台币199元（约合人民币46元）。</p><h3>Xbox台服订阅服务价格调整表（时间、原价、现价、降幅）</h3><p><img class="lazy" data-height="789" src="https://i1.max-c.com/imgx/2022/01/18/07055280e7044bd3a197a43fadbc9ec81642486933.jpg" data-width="311" referrerpolicy="no-referrer"></p><p>官方表示，Xbox会定期评估Xbox Game Pass 及Xbox Live Gold在各个市场当地的价格，以为各地区的会员及玩家带来最好的服务。这些价格调整会根据每个市场的情况而订定，也是微软去年底开始在一些地区进行价格调整的延续。</p><p>其中Xbox Game Pass Ultimate包含Xbox Live Gold会员的权益、最先进的多人游戏网路、以及上百款适用于主机与PC的高品质游戏，更包含EA Play会员。</p><p><img class="lazy" data-height="525" src="https://i1.max-c.com/imgx/2022/01/18/a2ae4a0df9a24860972b2e0906c5dd251642486934.jpg" data-width="1000" referrerpolicy="no-referrer"></p><blockquote><p>来源：游侠网</p></blockquote>
