<p class="MsoNormal">去年微软推出<span lang="EN-US">Win11</span>系统时，<span lang="EN-US">TPM</span>安全模块以及<span lang="EN-US">Intel 8</span>代酷睿<span lang="EN-US">/AMD</span>锐龙<span lang="EN-US">2</span>代及以上的硬件要求引发了网友不满，这提高了系统使用的门槛，现在来看微软是打算继续推这个标准下去了，下版<span lang="EN-US">Windows Server</span>系统也会如此。</p>
<p class="MsoNormal">微软前几天发布了<span lang="EN-US">Windows Server vNEXT Build 25057</span>系统，这就是<span lang="EN-US">Win11</span>版的<span lang="EN-US">LTSC</span>长期服务版系统，目前是测试中。</p>
<p class="MsoNormal">有网友在安装<span lang="EN-US">Windows Server vNEXT Build 25057</span>系统时发现了问题，<span style="color:#ff0000;"><strong>系统会提示硬件兼容性，其中主要是两个问题，一个是支持<span lang="EN-US">TPM</span>，一个是<span lang="EN-US">CPU</span>兼容，也是<span lang="EN-US">Intel 8</span>代酷睿<span lang="EN-US">/AMD</span>锐龙<span lang="EN-US">2</span>代及以上。</strong></span></p>
<p class="MsoNormal">这倒是不让人意外，因为<span lang="EN-US">Windows Server vNEXT Build 25057</span>就是基于<span lang="EN-US">Win11</span>系统开发的，硬件要求一样很正常。</p>

<p class="MsoNormal" style="text-align: center;"><img alt="不止Win11 下版Windows Sever系统也将强制要求TPM及CPU兼容" h="399" src="https://img1.mydrivers.com/img/20220219/db7490a4-1136-46d6-92e8-c1fa5b38fcc0.png" style="text-align: center; border: 1px solid black;" w="600" referrerpolicy="no-referrer"></p>
<p class="MsoNormal">按照<span lang="EN-US">Win11</span>系统的情况来看，用户也应该可以绕过这些限制强制安装系统，但这也可能导致一些更新无法升级，微软并不乐见不达标的系统升级新系统的。</p>
<p class="MsoNormal"><strong>好在<span lang="EN-US">Windows Server</span>新系统距离最终发布还有<span lang="EN-US">2</span>年时间，希望到时候用户的设备都能达标吧。</strong></p>
