<p>1月14日消息，Redmi即将在印度推出Note系列新品Redmi Note 11S，现在xiaomiui.net曝光了Redmi Note 11S的渲染图。</p>
<p>如图所示，Redmi Note 11S与Redmi Note 11造型基本一致，二者都是<span style="color:#ff0000;"><strong>中置挖孔屏方案、矩阵相机。</strong></span></p>
<p>据爆料，Redmi Note 11S代号miel，型号为2201117SI和2201117SG，是Redmi面向入门机市场打造的一款4G设备，预计会搭载联发科处理器，型号未知。PS：上一代Redmi Note 10S搭载联发科Helio G95。</p>
<p>此外，Redmi Note 11S后置1.08亿像素主摄、800万超广角、200万景深和200万微距四摄，其中<strong>主摄为三星ISOCELL HM2，这颗sensor曾被应用到Redmi超大杯旗舰Redmi K40 Pro+上。</strong></p>
<p><strong>它拥有1/1.52英寸大底，单位像素面积为0.7μm，</strong>可以通过像素九合一输出1200万像素图片。</p>
<p>该机有望在2月份发布。</p>
