<p data-vmark="f52a">11 月 24 日消息，亚马逊云科技宣布通过与光环新网和西云数据的合作，在中国区域（北京与宁夏）推出 Babelfish for Amazon Aurora PostgreSQL，客户无需或只需很少的代码更改，便可直接在 Amazon Aurora PostgreSQL 上运行 Microsoft SQL Server 应用。</p><p data-vmark="4460"><img src="https://img.ithome.com/newsuploadfiles/2021/11/2cb6fc78-9f1b-4708-b2f0-9d901c06a18e.jpg" w="630" h="205" title="亚马逊云科技 Babelfish for Amazon Aurora PostgreSQL 已在中国区域正式可用" width="630" height="205" referrerpolicy="no-referrer"></p><p data-vmark="7d05">这是亚马逊云科技在数据库迁移服务方面，在现有的数据库架构转换、数据迁移服务之上，进一步提供应用程序迁移服务，帮助客户加速向云上数据库的平滑迁移。</p><p data-vmark="bd7d">与此同时，亚马逊云科技提供多种迁移服务帮忙客户迁移数据库，包括 Amazon Schema Conversion Tool 可转换数据库架构， Amazon Database Migration Service 可迁移数据，新发布的 Babelfish for Amazon Aurora PostgreSQL 则可帮助客户迁移 SQL Server 应用程序。</p><p data-vmark="6a0f">目前，Babelfish for PostgreSQL 源代码现已在 Github 开源。中国用户还可以通过亚马逊云科技 Database Freedom 项目，了解如何从传统商业数据迁移至云数据库。</p>
