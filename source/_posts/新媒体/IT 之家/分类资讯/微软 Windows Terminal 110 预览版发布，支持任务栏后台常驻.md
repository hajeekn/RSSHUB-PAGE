<p><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 7 月 15 日消息 微软 Windows Terminal <span class="accentTextColor">1.10 预览版</span>于今日发布，带来了下拉菜单的命令行、任务栏后台常驻等功能。</p><h2>下拉菜单的命令行</h2><p>微软表示，下拉菜单的“反馈”按钮几乎没什么人用，因此把它换成了更实用的<span class="accentTextColor">命令行功能</span>，命令行也可以使用快捷键 Ctrl+Shift+P 打开。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/33f53fce-1190-4c39-aec9-17e51d7a4fc6.gif" w="1440" h="810" title="微软 Windows Terminal 1.10 预览版发布，支持任务栏后台常驻" width="1440" height="461" referrerpolicy="no-referrer"></p><h2>任务栏后台常驻</h2><p>Windows Terminal 1.10 预览版终于支持了<span class="accentTextColor">后台常驻</span>，用户可选择将其保留后台，任务栏右下角的系统托盘会出现 Windows Terminal 预览版图标，点击图标或者使用快捷键 Win+` 都可以快速打开终端。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/704a0db0-34ea-4304-a01e-acd9c805457f.png" w="585" h="98" title="微软 Windows Terminal 1.10 预览版发布，支持任务栏后台常驻" width="585" height="98" referrerpolicy="no-referrer"></p><h2>粗体文本</h2><p>Windows Terminal 1.10 预览版已支持<span class="accentTextColor">显示粗体文本</span>，官方表示将在未来为其添加设置功能，用户可自行配置。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/7da18247-7e20-45bf-ad54-2c62581605d5.png" w="1326" h="750" title="微软 Windows Terminal 1.10 预览版发布，支持任务栏后台常驻" width="1326" height="464" referrerpolicy="no-referrer"></p><h2>设置 UI 更新</h2><p>在 1.8 预览版本中，由于与 JSON 片段扩展产生的一些架构冲突，开发团队决定从设置 UI 中移除 base layer 页面。在 1.10 版本中，<span class="accentTextColor">base layer 页面回来了</span>，官方将设计一个扩展页面来帮助用户管理您的 JSON 片段扩展。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/f69edef7-02a2-48c4-94eb-58173602009a.png" w="1440" h="871" title="微软 Windows Terminal 1.10 预览版发布，支持任务栏后台常驻" width="1440" height="496" referrerpolicy="no-referrer"></p><p>Windows Terminal 1.10 预览版还允许用户<span class="accentTextColor">自定义快捷键</span>，只要与软件默认的几个快捷键不冲突就好。</p><p><img src="https://img.ithome.com/newsuploadfiles/2021/7/27547157-d191-4bf2-9f84-52b4d30dc3eb.png" w="1035" h="654" title="微软 Windows Terminal 1.10 预览版发布，支持任务栏后台常驻" width="1035" height="518" referrerpolicy="no-referrer"></p><p>此外，Windows Terminal 1.10 预览版可以在设置中<span class="accentTextColor">选择显示语言</span>了，用户可以像使用其他软件一样，选择中文显示。</p><p>IT之家了解到，Windows Terminal 1.10 预览版还修复了一系列崩溃问题，这里就不多介绍了，大家可以在 <a href="https://github.com/microsoft/terminal/releases" target="_blank">GitHub 页面</a>手动下载安装。</p>
