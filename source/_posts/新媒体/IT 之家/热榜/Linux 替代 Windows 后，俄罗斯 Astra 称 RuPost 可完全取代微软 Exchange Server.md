<p data-vmark="fb55"><a class="s_tag" href="https://www.ithome.com/" target="_blank">IT之家</a> 11 月 27 日消息，据 mspoweruser 报道，软件世界正变得越来越分散，为其他公司提供了机会，以填补通常的软件巨头的角色。</p><p data-vmark="f2c3">RuPost 就是这种情况，据称它是俄罗斯市场上微软 Exchange Server 的 1:1 替代品。</p><p data-vmark="1847"><img src="https://img.ithome.com/newsuploadfiles/2021/11/f774acc2-1472-4021-829e-278a6eee0025.jpg@s_2,w_820,h_448" w="926" h="506" title="Linux 替代 Windows 后，俄罗斯 Astra 称 RuPost 可完全取代微软 Exchange Server" srcset="https://img.ithome.com/newsuploadfiles/2021/11/f774acc2-1472-4021-829e-278a6eee0025.jpg 2x" width="926" height="448" referrerpolicy="no-referrer"></p><p data-vmark="c5b1">IT之家获悉，RuPost 是由俄罗斯 Astra 公司集团开发的，并声称完全实现了微软 Exchange Sever 的功能。</p><p data-vmark="192d">该解决方案提供邮件、日历、联系人和地址簿搜索，还提供迁移工具和模板，让公司轻松从 Exchange 转移到 RuPost。</p><blockquote><p data-vmark="9bfb">“首先，我们专注于解决拥有大规模基础设施的组织所面临的问题：一个企业通信系统将允许数以万计的用户在一个高负荷和地理分布的 IT 基础设施中进行互动，”产品和技术总监 Sergey Orlik 解释说。</p></blockquote><p data-vmark="acb3">该公司提供了一个全堆栈，还用 Astra Linux 系统取代了 Windows。它还创建了 ALD Pro，作为微软 Active Directory（活动目录）的替代品。</p><blockquote><p data-vmark="1c80">他说：“我们正在发展和加强 Linux 生态系统，并创建我们自己的互连解决方案的生态系统：一个完整的软件产品和服务堆栈，客户可以从供应商那里获得一站式服务模式。”</p></blockquote><p data-vmark="ff27">ALD Pro 将于 2021 年 12 月发布，而 RuPost 将于 2022 年第一季度末上市。</p><p data-vmark="502d">该解决方案可能会在政府设施中找到它的第一个应用，在那里，本地解决方案是强制性的，尽管 Astra 指出，如果微软的解决方案是绝对必要的，公司仍然可以申请例外。</p>
