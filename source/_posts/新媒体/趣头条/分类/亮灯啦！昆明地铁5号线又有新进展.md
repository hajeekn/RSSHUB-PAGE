<p>好消息！</p><p>昆明地铁5号线最新进展来了</p><p>昆明地铁5号线预计今年上半年开通运营，目前，建设现场如火如荼。3月1日，记者在5号线穿金路站看到，车站各出入口立柱已立好，其中B口、C口立柱已通电亮灯。穿金路站</p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/0662c221de0b174b4a18249ed0af0804.jpeg" data-size="480,639" referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/3ccce7c9bd3989cfba101800162de83d.jpeg" data-size="480,639" referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/477656711c21f0283db51500bac2f9a8.png" data-size="480,300" referrerpolicy="no-referrer"></p><p>穿金站位于穿金路与新兴巷路交叉口，该站3个出入口位于米轨两旁，非常醒目，位置分布参考上图绿圈。</p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/38166530b69407994b937a2ecf6bedda.jpeg" data-size="480,360" referrerpolicy="no-referrer"><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/994579a1aaff02d2fb7f6f9a239eb0f5.jpeg" data-size="480,360" referrerpolicy="no-referrer"></p><p>昆明地铁5号线全长26.45公里，共设有22座车站。贯穿盘龙区、五华区、西山区、滇池度假区和官渡区，途径世博园片区、圆通公园片区、会展中心片区，串联起世博园、圆通山、翠湖、滇池等昆明极具代表性的旅游景区。5号线开通后，可与已开通运营的1、2号线首期、3号线、4号线换乘。</p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/544b5a52fe7bdc6943a9a800b0c6eeb5.png" data-size="480,646" referrerpolicy="no-referrer"></p><p>地铁5号线途经站点</p><p>世博园站 → 白龙寺站 → 石闸站 → 穿金路站 → 火车北站 → 圆通山站 → 华山西路站 → 五一路站 → 弥勒寺站 → 严家地站 → 金兰路站 → 福海站 → 广福路西口站 → 大坝站 → 迎海路站 → 渔户村站 → 兴体路站 → 庄家塘立交桥站 → 金家河站 → 福保站 → 滇池国际会展中心站 → 宝丰站</p><p>地铁5号线换乘站</p><p>火车北站：与2、4号线换乘</p><p>五一路站：与3号线换乘</p><p>弥勒寺站：与1号线西北延换乘</p><p>滇池国际会展中心站：与2号线换乘</p><p>宝丰站：与2号线换乘</p><p><img src="https://cors.zfour.workers.dev/?http://static.1sapp.com/qupost/img/2022/03/02/726cfef99f956908163a8b071be68215.jpeg" data-size="480,320" referrerpolicy="no-referrer"></p><p>来源：无线昆明、昆明出行</p>
