<strong>Adobe 刚刚推出了名为 Creative Cloud Express 的移动兼 Web 应用，以期将该公司繁杂的创意云套件和 Acrobat PDF 工具中的某些高票功能，整合到这个单一的应用程序中。</strong>无论是快速创建社交媒体帖子、宣传海报、或视频内容，Creative Cloud Express 都可作为 Adobe Spark 套件的一个不错替代。<br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1214/cddc3161331fd6a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1214/cddc3161331fd6a.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Adobe <a href="https://www.adobe.com/express/" target="_self">官网</a>）</p><p>得益于内置的丰富模板和图像等资源，Creative Cloud Express 的访问体验，显然比单独的创意云应用要轻松得多。</p><p>除了免费版，Adobe 还提供了月费 9.99 美元的付费选项，主要包括了额外的功能、以及更加繁杂的模板库。</p><p>当然，如果你已经是创意云的订户，那 Creative Cloud Express 也可供你自由使用。</p><p><img src="https://static.cnbetacdn.com/article/2021/1214/b399b37077129f8.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p>感兴趣的朋友，现可通过<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a> App Store、谷歌 Play 商店、以及<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a> Microsoft Store 获取免费版。如果觉得移动 App 还不够，那你也可通过 Web 渠道来体验。</p><p>据悉，Creative Cloud Express 的总体理念，就是为非专业人士提供实现其愿景所需的工具。Ashley Still 指出，近年来，公司已经见证了非专业用户的大量增长。</p><p>尴尬的是，许多人最初都错误预估了他们对完整版创意云应用程序的功能需求和技能要求。有鉴于此，Adobe 才特地融合出了可通过更加便捷的方式来完成同样任务的 Creative Cloud Express 。</p><p><img src="https://static.cnbetacdn.com/article/2021/1214/a26d06cb85d61e8.gif" alt="2.gif" referrerpolicy="no-referrer"></p><p>Creative Cloud Express 汇集了 Adobe 广泛的 Web / 移动应用程序，以及完整的创意云技术。无论是 Photoshop 图片处理、还是视频编辑，该套件都可轻松满足。</p><p>更确切地说，Creative Cloud Express 更适合那些“专注于结果”、而非专注于过程的用户。因为他们并不希望从一页白纸开始，而是充分利用拥有 1.75 亿图像的强大的 Adobe Stock 资源库。</p><p>这部分用户群体也不想了解如何创建字体，而是直接从 20000 款 Adobe 字库中跳出他们所需的那一个。更不想横跨多款 Adobe 生产力应用去搞定一份传感，最终以 PDF 的形式来输出。</p><p><img src="https://static.cnbetacdn.com/article/2021/1214/b2cd1ea7f92e705.gif" alt="3.gif" referrerpolicy="no-referrer"></p><p>显然，Creative Cloud Express 能够很好地帮助这部分用户在一处轻松完成这些需求。比如在庞大的模板库中挑中了一款之后，可立即通过相关工具来移除背景、或为图像套上 Photoshop 的风格滤镜。</p><p>此外得益于与 Creative Cloud Libraries 资源库的集成，用户还可充分调用同事或其他人创建的资源，然后在 Creative Cloud Express 应用程序中轻松调用。</p><p>其它工具包括可将视频转换为 GIF 动图、将文档转换为 PDF 格式等。不过最有趣的是，还是 Adobe Stock 的集成 —— 虽然单独并不免费，但 Creative Cloud Express 还是获得了“有限的访问”。</p>
