<strong>在转向为期四周的更新周期后，Google 刚刚跳过了 Chrome OS 95、直接放出了 Chrome OS 96 版本。</strong>另一方面，该公司有向教育和企业 Chromebook 客户提供六个月的长期支持（LTS）通道。功能方面，Chrome OS 96 特别强调了相机体验的改进。配备了前后置摄像头的 Chrome OS 设备，现可通过相机应用内置的文档扫描仪功能，将捕获的内容转换为 PDF 或 JPEG 格式。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1201/f02ee6dccf0452e.jpg" referrerpolicy="no-referrer"></p><p>在“二维码”旁边，你将看到新增的这个“扫描”选项卡。启用后，相机应用会自动检测边缘，当然用户也可手动调节。如果你之前已经用过 Chrome OS 94，或许并不会对本次更新感到陌生。</p><p><a href="https://static.cnbetacdn.com/article/2021/1201/6222b2d0bb47e72.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1201/6222b2d0bb47e72.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>与此同时，使用兼容款外置摄像头的 Chrome OS 设备用户，现可借助 Pan-Tilt-Zoom 来控制角度和裁切。而使用相机应用的新 D-pad（位于左侧栏）进行调整的方式，在各种用例中也得到了沿用。</p><p><a href="https://static.cnbetacdn.com/article/2021/1201/b5f28d56abae1da.jpeg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1201/b5f28d56abae1da.jpeg" alt="2.jpeg" referrerpolicy="no-referrer"></a></p><p>展望明年，Chrome OS 相机应用还将迎来一款 GIF 制作工具，以便用户轻松录制长达 5 秒的片段。此外 Google 也在测试为虚拟助理集成“拍照”、“视频录制”和“自拍”等召唤功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1201/463d721b0221cec.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1201/463d721b0221cec.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>至于 Chrome OS 用户期待已久的“附近分享”（Nearby Share）功能，现也能够在 Android 应用程序、及其分享列表中访问。此前，该功能仅限于文件应用程序、渐进式 Web 应用（PWA）、以及系统应用。</p><p>其它方面，Chrome OS96 在“设置 → 应用”中引入了一个新的通知页面（此前仅在快速设置中提供），以便用户开启或关闭。</p><p><a href="https://static.cnbetacdn.com/article/2021/1201/1e92533574c25a6.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1201/1e92533574c25a6.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>同样，“应用管理”列表中的 Apps，新增了一个“打开支持的链接”部分，可帮助用户确定网址（URL）是打开已安装的客户端、还是在 Chrome 浏览器中呈现。</p><p>最后，壁纸应用最近迎来了更大的设计变更、引入了更大的网格 UI、并且能够通过启动器图标访问（而不是只能在桌面上点击鼠标右键）。</p>
