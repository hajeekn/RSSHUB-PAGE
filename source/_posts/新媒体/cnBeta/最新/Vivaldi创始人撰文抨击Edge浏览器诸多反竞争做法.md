Windows 11 的设置和广告迫使用户将 Microsoft Edge 浏览器作为默认浏览器，这引起了 Vivaldi 浏览器的反感。<a href="https://vivaldi.com/blog/microsoft-back-to-its-old-tricks-to-get-an-edge-on-the-competition/" target="_blank"><strong>该公司的联合创始人 Jon von Tetzchner 撰写了一篇博文</strong></a><strong>，讨论微软严重的反竞争做法。</strong><br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1213/c08c7f593ec7a4a.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1213/c08c7f593ec7a4a.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">相比较其他基于 Chromium 的浏览器，Microsoft Edge 在 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 平台上是有主场优势的。而近期<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>采取了一些激进的做法引起了用户和同行的强烈不满。</p><p style="text-align: left;">最新的尝试之一是关于 Microsoft Edge 浏览器内的一个新横幅，当用户试图下载 Chrome 浏览器时，该横幅称 Chrome 是 2008 年的浏览器。此外，微软还让人难以将 Vivaldi 或 Chrome 设置为 Windows 11 的默认浏览器。Vivaldi 显然对微软说服用户将 Chromium Edge 作为 Windows 11 的默认浏览器的最新尝试不满意。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1213/f53f9fdcdd74493.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1213/f53f9fdcdd74493.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Vivaldi 联合创始人兼 CEO Jon von Tetzchner 表示：“微软的举动似乎很绝望。而且很熟悉。很明显，他们不希望你使用其他浏览器”。Vivaldi 呼吁微软的一些做法阻碍了其他浏览器的业务，包括 Vivaldi。例如，如果你使用微软 Edge 在 Bing 上搜索 Vivaldi，你会被要求继续使用 Chromium Edge 以获得"快速、安全和现代的网络体验"。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1213/e91efaafe930ad7.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1213/e91efaafe930ad7.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/1213/c8705bdfb1194ea.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1213/c8705bdfb1194ea.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">如果你无视建议在Windows上安装Vivaldi，Windows 10会再次要求你保留Microsoft Edge，因为它更好。但 Jon von Tetzchne 认为这在 Windows 11上会变得更加糟糕。</p><p style="text-align: left;">正如Vivaldi所强调的，Windows 11让改变默认浏览器这样的基本事情变得困难。与Windows 10不同，Windows 11要求你首先选择浏览器，并将其指定为所有文件扩展的默认浏览器。</p><p style="text-align: left;">你需要为各种网络相关的文件扩展名手动选择第三方浏览器，包括HTTP、HTTPs、htm、HTML或文件。而如果你真的试图改变所有这些文件扩展名的默认值，Windows 11会显示又一个弹出窗口，强调Edge的功能。</p><p style="text-align: left;">Vivaldi 创始人说：“他们甚至提出要通过他们的微软奖励计划向你支付使用该浏览器的费用。这不是一个开发卓越浏览器的自信公司的行为。这是一家公司公开滥用其强大的地位，推动人们使用其劣质产品的行为，仅仅是因为它可以”。</p><p style="text-align: left;">Vivaldi创始人告诉用户，在社交媒体平台上投诉Windows 11的反竞争行为。该公司还建议美国和欧洲的用户给反垄断监管机构写信或打电话，对微软的这些反竞争行为进行调查。</p>   
</div>
