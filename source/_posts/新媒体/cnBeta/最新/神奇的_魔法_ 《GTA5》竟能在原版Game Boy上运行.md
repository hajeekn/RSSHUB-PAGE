名为 Sebastian Staacks 的网友是一位程序员，同时也是一位优秀的传统修理工，他的博客“There Oughta Be（理所应有）”充斥着各种各样的技术“奇迹”，从 LED 立方体到自制的草坪自动灌溉系统。最近，<strong>Staacks 非常地怀旧，首先是制作了 Wi-Fi Game Boy 卡带，之后又让《GTA5》以及《DOOM》在这款已经近半个世纪历史的掌机上运行。</strong><br>
 <p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2022/0124/e9105c8f3cb1d96.jpg"><img data-original="https://static.cnbetacdn.com/article/2022/0124/e9105c8f3cb1d96.jpg" src="https://static.cnbetacdn.com/thumb/article/2022/0124/e9105c8f3cb1d96.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">“几周前，我创建了一个带有内置 WiFi 的 Game Boy 卡带。”Staacks 写道：“现在我让它在未经修改的原始 Game Boy 上以全分辨率 20 FPS 云直播视频和玩游戏。”</p><p style="text-align: left;">他在自己的博客中展示了更加详细的技术情报，同时也发布了一个更通俗的解释视频。在视频中你可以看到这些现代游戏确实是在这款仅有黑白的掌机上运行，真的就如同“魔法”一般。</p><p style="text-align: left;">技术演示视频：</p><p style="text-align:center"><iframe height="498" width="640" src="https://player.youku.com/embed/XNTIwNDIxODQ5Ng==" frameborder="0"></iframe></p><p style="text-align: left;">原版 Game Boy 的屏幕显然无法完全胜任所有工作，特别是考虑到对比度问题，黑白和灰色的区间并不明显。之后他又在 Analogue 公司生产的高清版 Game Boy —— “Pocket” 主机上展示了自己的 Wi-Fi 卡带。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0124/efd5c635521dda0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">世界上所有的系统有必须要能运行的三样东西：《Bad Apple》、《孤岛危机（Crysis）》以及原版《毁灭战士（Doom）》。自然，一位他的粉丝展示了自己的 Game Boy 运行《毁灭战士》的样子。然而考虑到这有的玩家甚至可以在《我的世界》中创造可以运行《毁灭战士》的电脑， Game Boy 可以运行这款游戏已经让人见怪不怪了。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0124/b7c2a5a03ff8a14.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">最后 Staacks 说道：“伴随着新的演示，许多其他的演示都无法再与之比较。我可以（让 Game Boy）显示任何东西，我可以控制任何可以在我的 PC 上显示和控制的东西。当然，这样的界面和图像几乎无法使用，因此一些专门的用法也可能会很有意思。同时我并没改动音频，这是我设计这款卡带时不幸的巨大失误。”</p>
