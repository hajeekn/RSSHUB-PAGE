作为 Valve 的 Wayland 合成器，Gamescope 有助于在开源的 Linux 平台上运行游戏。<strong>最新消息是，它已添加对 AMD FidelityFX Super Resolution（简称 FSR）技术的支持。</strong>据悉，Gamescope 基于 Valve 的 SteamOS 微合成器（文件名 steamcompmgr）打造，可为 Wayland 交付最让人感到满意的游戏性能。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0202/5142af0f3939270.jpg" alt="FSR.jpg" referrerpolicy="no-referrer"></p><p>相关流程包括在标准的 Linux 桌面环境中作为嵌套合成器来运作，且 Gamescope 已支持整数缩放和其它组件集成。</p><p>现在，它又提供了对 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 开源的 FSR 方案的支持，致力于为下一代游戏玩家和用户带来更佳的图形体验。</p><blockquote><p>Phoronix <a href="https://www.phoronix.com/scan.php?page=news_item&px=Gamescope-AMD-FSR" target="_self">指出</a>，昨日在 GitHub 上提交的最新合并工作，已经为 Gamescope 引入了对 AMD FSR 的支持。</p><p>Gamescope 现可使用 -U 启动参数来启用 FSR 支持，辅以 --fsr-sharpness 的锐度支持和用于变更 FSR 放大 / 锐度行为的快捷键。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0202/167d4ae7a02cc0d.jpg" alt="FSR Logo.jpg" referrerpolicy="no-referrer"></p><p>WCCFTech <a href="https://wccftech.com/amd-fidelityfx-super-resolution-support-comes-to-valve-gamescope-compositor/" target="_self">补充道</a>：为 Linux 平台上的 Vulkan 项目和 Gamescope 构建做出了重要贡献的程序员 Georg Lehmann，也为 Proton / FSR 支持付出了相当大的心力。</p><p>Valve 已确保新推出的 Steam Deck 掌机具有跨平台支持特性，以提起游戏玩家们的极大兴趣，而新传闻又提到了 Gamescope 支持。</p><p>最后，其它 Linux 桌面发行版用户，也将受益于 AMD FSR 技术和 Gamescope 合成器带来的相关改进。</p>
