Xbox负责人Phil Spencer称Xbox Game Pass当前的状态已经是“可持续了”，而且将来只会变得更强大。在接受外媒Axio采访时，Spencer被问到有人说Game Pass不可持续，因为玩家不用花60美元就能玩《星空》一类的大作。<br>
 <p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1116/87f1e54252f7c57.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1116/87f1e54252f7c57.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1116/87f1e54252f7c57.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">Spencer回应说：“你可以算算Game Pass这笔账。我猜你不知道我们有多少订阅玩家，也不知道每个订阅玩家花多少钱，但你可以做出一些相当明智的决定。纯粹是对我们认为Game Pass最终可能是什么进行计算--你可以对业务的每个部分都这样做。不过，Game Pass绝对是可持续的。”</p><p style="text-align: left;">Spencer补充说Game Pass的增长是Xbox策略的一部分，不是公司的唯一专注点。</p><p style="text-align: left;">“Game Pass不是Xbox唯一的增长点，也不是我们唯一的专注点。不过单独来说，它当前的状况非常可持续。我知道外面有很多人说我们正在烧钱，是为了将来赚大钱。不是这样的，Game Pass现在是非常、非常可持续的，它会继续增长。”</p><p style="text-align: left;">Xbox Game Pass 2017年6月推出。在7月举行的Q4财报上，微软CEO纳德拉表示Xbox Game Pass正在快速地成长，订阅用户比非订阅用户多玩了40%的游戏，消费也比后者多了50%。</p><p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1116/3b5aa120dd34b74.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1116/3b5aa120dd34b74.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1116/3b5aa120dd34b74.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">此前，前SIE美国CEO Shawn Layden曾断言Xbox Game Pass不可持续。如果想要可持续，至少需要5亿订阅用户。原因在于当今游戏开发成本越来越高，通过订阅模式发售3A大作将需要相当大体量的玩家才能回本。</p><p style="text-align: left;">“你很难将一个1.2亿美元的游戏推出在月费9.99美元的订阅服务上。你必须要有5亿用户才能开始收回投资。所以为了壮大这一基数，你必须先亏损。”</p><p style="text-align: left;">值得一提的是，今年5月，The Verge资深记者&高级编辑Tom Warren在Twitter上透露Xbox Game Pass尚未实现盈利。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1116/64167c20af0d84a.jpg" referrerpolicy="no-referrer"></p><p style="text-align: left;">Xbox Game Pass可能是微软这一世代最大的王牌，该服务为用户提供数百款游戏，每月订阅价格也非常合理。微软的游戏生态系统主管Sarah Bond在接受GQ杂志采访时透露，微软对Game Pass的最初设想并不是这样，而是一个租赁服务。</p><p style="text-align:center"><a target="_blank" href="https://static.cnbetacdn.com/article/2021/1116/6686676311e1cd4.jpg"><img data-original="https://static.cnbetacdn.com/article/2021/1116/6686676311e1cd4.jpg" src="https://static.cnbetacdn.com/thumb/article/2021/1116/6686676311e1cd4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">XGP项目代号为Arches，一开始的设想是游戏租赁服务。然而微软团队注意到了当前市场游戏销售周期的转变，决定改变这一计划，变成了Netflix这样的订阅模式。</p><p style="text-align: left;">Bond告诉GQ：“一个游戏收入的75%过去都是在两个月内完成的，现在普遍是超过两年时间。”</p><p style="text-align: left;">此外报道指出，Xbox团队当初带着Game Pass概念找发行商时普遍被拒绝。</p><p style="text-align: left;">Bond回忆说：“当时发行商的反映就是：没门，Game Pass会让游戏贬值。”</p><p style="text-align: left;">微软团队为了说服发行商，他们用更老的游戏进行试验。现如今，Xbox Game Pass已经取得了相当大的成功，微软通过收购工作室，让每个季度都能有新的第一方游戏登陆Xbox Game Pass。</p>
