在北京时间 3 月 9 日凌晨 2 点的春季新品发布会上，苹果推出了 27 英寸 @ 5K 规格的 Studio Display 。<strong>作为 Pro Display XDR 的较入门版本，起售价低至 1599 美元、并且提供了 VESA 安装和“可调支架”的选项。</strong>如果你习惯了在黑暗环境中开展视频编辑工作，基础型号已经相当够用。但若身处具有明亮灯光 / 开放式隔间的媒体中心，眩光就是个绕不开的话题。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0309/c6bd7aff5e74fb3.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0309/c6bd7aff5e74fb3.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（来自：Apple <a href="https://www.apple.com/shop/buy-mac/mac-studio-display/nano-texture-glass-tilt-and-height-adjustable-stand" target="_self">官网</a>）</p><p>好消息是，<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>还为 Studio Display 提供了纳米纹理玻璃的选项，能够显著减少在屏幕上产生的眩光，但售价也上跳到了 1899 美元。</p><blockquote><p>支架方面，除了仅可调节 30° 俯仰角的版本，Studio Display 也可选配支持高度调节的高级型号，那样总价也会增加到 1999 美元。</p><p>基于此，如果你的工作环境比较明亮、且桌面高度较低，那在选配高度可调支架 + 纳米纹理防眩光玻璃的情况下，Studio Display 的入手价格就高达 2299 美元。</p></blockquote><blockquote></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0309/63e1e3f7ece4845.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0309/63e1e3f7ece4845.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>另外需要注意的是，苹果仅在包装箱内附带了一条 3 英尺（约 0.9 米）长的雷电线缆。</p><blockquote><p>如需更长的布置，新款 1.8 米雷电 4 Pro 线缆的报价为 <a href="https://www.apple.com/shop/product/MN713AM/A/thunderbolt-4-pro-cable-18-m" target="_self">129 美元</a>（约 815 RMB），‘即将推出’的 3 米版本则是 <a href="https://www.apple.com/shop/product/MWP02AM/A/thunderbolt-4-pro-cable-3-m" target="_self">159 美元</a>（约 1004 RMB）。</p><p>作为参考，贝尔金（Belkin）2 米雷电 3 线缆在亚马逊上的报价仅为 <a href="https://www.amazon.com/dp/B072KPBKS3/" target="_self">64.99 美元</a>（约 411 RMB）。</p></blockquote><p><a href="https://static.cnbetacdn.com/article/2022/0309/663b5ea06ab4797.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0309/663b5ea06ab4797.png" alt="3.png" referrerpolicy="no-referrer"></a></p><p>最后，如果你想要支持俯仰 / 高度调节、配有纳米纹理防眩光玻璃的 Studio Display 显示器，再加上尚未发布的 3 米雷电 4 线缆，那总计入手成本就高达 2458 美元（将近 1.6 万 RMB）—— 较最入门的型号高出 500 美元。</p>
