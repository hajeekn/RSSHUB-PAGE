本周四，一家“为消费者提供无缝数字体验”的软件公司 <strong>SpaceTime3D，在美国德克萨斯州西区地方法院提起了诉讼，指责苹果的某些图形用户界面功能侵犯了该公司的 3D UI 专利。</strong>在 SpaceTime3D 主张的三项存在争议的美国专利中，其中一项可追溯到 2007 年。<br>
<p><img src="https://static.cnbetacdn.com/article/2022/0212/a2bf5d05585a838.png" referrerpolicy="no-referrer"></p><p>原告方在诉状中指出：</p><blockquote><p>苹果的应用切换器、以及 Safari 浏览器的标签视图功能，侵犯了该公司的相关专利。</p><p>因其涵盖了一种通过在 3D 空间中显示内容，以改善小型设备用户体验的方法。</p><p>用户可借助这些功能，轻松查看 / 切换多个选项卡、或打开应用程序，并且适用于 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a> / Apple Watch 等设备。</p></blockquote><p>与之前的专利诉讼相比，周四的这起案件，并非由非执业实体提起。<a href="https://appleinsider.com/articles/22/02/11/spacetime3d-lawsuit-claims-app-switcher-safari-tabs-infringe-on-patents" target="_self">Apple Insider</a> 指出：</p><blockquote><p>SpaceTime3D, Inc. 有在维护一款应用程序，允许用户在选项卡式的三维堆栈中显示页面，从而带来直观、有趣、交互式的 Web 浏览体验。</p><p>在 2008 年的消费电子展（CES）上，该公司的技术也得到了重点关注，并被《San Jose Mercury News》和 TC 等新闻媒体所报道。</p></blockquote><p>诉讼宣称苹果通过新闻报道意识到了该技术，并补充公司创始人 Ezra Bakhash 曾向一位未透露姓名的苹果高管展示了这项技术，而后者又将信息传递给了苹果内部的某个团队。</p><p>据说另一位苹果代表就 SpaceTime3D 的技术感到“震惊”，且表示该平台能够节省大量的时间开销。报道称该高管让 Bakhash 保持沟通，且他“肯定会传播这个消息”。</p><p>由于被苹果提前知悉了该专利技术，SpaceTime3D 起诉该公司故意侵犯其知识产权，且这家库比蒂诺科技巨头甚至试图为这项技术申请专利。</p><p>庆幸的是，因 SpaceTime3D 持有现有专利，苹果的计谋才没有得逞。综上所述，原告方希望陪审团裁定苹果侵犯了自家专利，对相关损害给予赔偿、并承担该案的法务支出。</p>
