<strong>由近日向 Dev 通道的 Insider 测试者们推送的 Windows 11 Build 22567 预览版本可知，微软正努力让 Windows Update 变得更加环保。</strong>具体说来是，系统将尽量安排在一天中的特定时段来安装更新，从而达到降低碳排放的目的。毕竟许多地区的电网都依赖于多种能量来源 —— 包括可再生能源和传统化石燃料。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0303/2b67dd10e779d67.png" referrerpolicy="no-referrer"></p><p>在升级到 Build 22567 编译版本后，只要有可能，<a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 就会在电网调用更多清洁能源（如风能、水电、太阳能）的情况下，优先安排在后台安装更新。</p><p>若顺利启用，Windows 11 就会在更新页面上显示一条信息，以便用户知晓系统正在努力尝试减少碳排放。</p><p>不过需要注意的是，这一功能选项并不一定适用于每一个用户（取决于已知的清洁能源信息）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0303/3335e879920bb46.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0303/3335e879920bb46.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">截图（来自：Windows <a href="https://blogs.windows.com/windows-insider/2022/03/02/announcing-windows-11-insider-preview-build-22567/" target="_self">Blog</a>）</p><p>微软解释称：“只有当你的 PC 已接入电源，且合作伙伴 powerMap 或 WattTime 提供了所在区域的碳强度数据时，这项功能才可被顺利启用”。</p><p>届时用户可在系统更新的设置页面上看到如上图所示的一条文本信息 ——“Windows Update 承诺帮助减少碳排放”。</p><p>至于这项功能将于何时正式向所有 Windows 11 用户推送，目前暂不得而知。此外用户仍可随时移步至“设置 → Windows Update”来手动检查并立即安装更新。</p>
