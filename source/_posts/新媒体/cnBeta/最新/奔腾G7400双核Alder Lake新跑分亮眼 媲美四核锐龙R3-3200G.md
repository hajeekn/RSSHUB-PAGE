几天前，WCCFTech 报道了英特尔入门级 Alder Lake 处理器（奔腾 G7400 / 赛扬 G6900）的基准测试成绩，可知其单核性能媲美高端 SKU 。<strong>现在，我们又见到了采用 2C / 4T 设计的奔腾 G7400，可知其在 Cinebench 基准测试中的表现，媲美 AMD 的锐龙 R3-3200G 四核竞品。</strong><br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1120/21efcaa56eb8a32.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">在保持相同内核 / 线程数的情况下，G7400 较前代产品领先 30% 。</p><p>规格方面，奔腾 G7400 拥有双核心 / 四线程，配备了 10nm ESF 工艺的 Golden Cove 高性能核心（P 核），但没有采用 Gracemont 节能核心（E 核）。</p><p><a href="https://static.cnbetacdn.com/article/2022/0117/32c933be8caa1ce.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0117/32c933be8caa1ce.jpg" alt="0.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">视频截图（来自：RandomGamingHD）</p><p>作为一款相当入门的双核处理器，其特点是拥有高达 3.7 GHz 的主频。热设计功耗（TDP）仅 46W，辅以 2.5MB L2 + 6MB L3 缓存，售价低至 64 美元（约 407 RMB）。</p><p><img src="https://static.cnbetacdn.com/article/2022/0117/e2fb240250128a9.png" alt="3.png" referrerpolicy="no-referrer"></p><p>周四的时候，@mate_mmder 在 Twitter 上分享了于 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 8（64-bit）操作系统平台上开展的 Cinebench R23 / R15 基准测试成绩。</p><p><a href="https://static.cnbetacdn.com/article/2022/0117/f03164824ff04a0.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0117/f03164824ff04a0.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p>可知 G7400 在 Cinebench R23 中获得了单线程 1396 / 多线程 3814 分的成绩，且 Cinebench R15 亦有 205 / 543 分的成绩。</p><p><a href="https://static.cnbetacdn.com/article/2022/0117/86bd1ab193f8dec.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0117/86bd1ab193f8dec.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>作为参考，WCCFTech 也分享了 Computerbase 的公共 Cinebench R23 基准测试数据库，以便大家比较 G7400 与其它几款同级 CPU 的水平高低。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=319129406&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">The New 12th Gen Intel Celeron G6900 Definitely Not an i9 Killer（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzE5MTI5NDA2LnNodG1s.html" target="_self">via</a>）</p><p>最让人感到欣喜的是，尽管 G7400 只拥有两个物理核心（2P + 0E），但它还是能够与较老的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 锐龙 R3-3200G 四核竞品一较高下。</p><p><img src="https://static.cnbetacdn.com/article/2022/0117/f2e509bfd0506de.png" alt="4.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Cinebench R23 横向对比</p><p>不过与基于 Zen 2 / Zen 3 设计的锐龙 4000G / 5000G 相比，奔腾 G7400 的 IPC 性能优势，就没有赛扬 G6900 那么显著了。</p><p>究其原因，或许是后者选用的华擎 BFB 测试平台解锁更高的功率限制。如果 G7400 也能够再飙高 1 GHz，我们大可期待它能够给出更亮眼的成绩。</p><p><img src="https://static.cnbetacdn.com/article/2022/0117/57bf5099f6c9d30.png" alt="5.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">Cinebench R15 横向对比（截图 via <a href="https://wccftech.com/intel-pentium-gold-g7400-dual-core-alder-lake-cpu-is-on-par-with-amds-ryzen-3-3200g-quad-core-in-cinebench-benchmark/" target="_self">WCCFTech</a>）</p><p>当然，就算是默认参数状态下，G7400 还是较上一代 G6400 具有高达 30% 的领先优势，有望为 100 没有以下的 CPU 市场带来更大的吸引力。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a
