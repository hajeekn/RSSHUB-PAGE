AMD 最近刷新了 Radeon RX 6000S 系列 RDNA 2 移动 GPU 阵容，让本就让人感到困惑的笔记本电脑显卡产品线变得更加复杂。<strong>不过随着新基准测试的泄露，我们终于可以明确知晓 M 和 S 系列 SKU 的主要区别 —— RX 6800S 的性能，较 RX 6800M 要落后 11% 左右 。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0217/bdcd14dc8955790.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0217/bdcd14dc8955790.png" alt="1.png" referrerpolicy="no-referrer"></a></p><p>市场定位方面，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> Radeon RX 6000M 系列属于标准 SKU，而新发布的 Radeon RX 6000S 系列专为轻薄 / 功耗而设计，所以性能差距也可以理解。</p><p><img src="https://static.cnbetacdn.com/article/2022/0217/b15e2b90986596f.png" alt="3.png" referrerpolicy="no-referrer"></p><p>不过更让人感到迷惑的，还是因为部件的规格有所不同。比如近日泄露的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://asus.jd.com/" target="_blank">华硕</a> ROG Zephyrus G14 游戏本，它就配备了 RX 6800S 移动 GPU 。</p><p>其具有 32 计算单元（CU）/ 2048 个流处理器，游戏提升频率可达 1975 MHz，辅以 8GB GDDR6 显存 @ 256GB/s 的带宽。</p><p><a href="https://static.cnbetacdn.com/article/2022/0217/14fae0f56b54b49.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0217/14fae0f56b54b49.png" alt="4.png" referrerpolicy="no-referrer"></a></p><p>相比之下，RX 6800M 具有 40 CU / 2560 个流处理器，游戏频率可达 2300 MHz，辅以 12GB GDDR6 显存 @ 386 GB/s 的带宽。</p><p>热设计功耗（TDP）方面，RX 6800S 为 100W，而 RX 6800M 可达 145W+ 。FP32 峰值性能方面，两者分别可达 11.78 / 9.22 TFLOPs 。</p><p><img src="https://static.cnbetacdn.com/article/2022/0217/6add9d4ea17ec05.png" alt="5.png" referrerpolicy="no-referrer"></p><p>在泄露的 Geekbench 5 OpenCL 基准测试中，可看到 RX 6800S 的得分为 82734 分、RX 6800M 则是 93373 的平均分。</p><p>在选购 6000M / 6000S 移动 GPU 的时候，还请注意 11% 左右的性能差距。</p><p><a href="https://static.cnbetacdn.com/article/2022/0217/ad0d2d595117602.png" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0217/ad0d2d595117602.png" alt="2.png" referrerpolicy="no-referrer"></a></p><p>另一方面，竞争对手英伟达家也有提供“Max-Q”衍生型号。与标准部件相比，其主要变化在于时钟频率和 TGP 功耗上。</p><p>从规格参数来看，一些人可能担心 RX 6800S 不及 TX 6650M XT 。但考虑到巨大的规格削减，其实际速度还是高出 RX 6700M 一截。</p>
