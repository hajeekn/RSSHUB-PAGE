<strong>作为 AMD 旗下首个采用全新 3D V-Cache 缓存技术的数据中心 CPU 产品线，该公司已证实霄龙（EPYC）Milan-X 处理器将于本月晚些时候到来。</strong>AMD 高级副总裁兼服务器部门总经理 Dan McNamara 在 Susquehanna Financial Group 年度技术会议上表示，EPYC Milan-X CPU 已在出样，预计可在流体力学计算、EDA 和各种类型的高级物理建模等计算工作负载中实现高达 50% 的性能提升。<br>
 <p><a href="https://static.cnbetacdn.com/article/2022/0226/4831edac132f716.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2022/0226/4831edac132f716.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（via <a href="https://wccftech.com/amd-confirms-epyc-milan-x-3d-v-cache-cpus-launching-later-this-month/" target="_self">WCCFTech</a>）</p><p>Dan McNamara 对霄龙 Milan-X 家族感到十分激动，得益于三倍的缓存数量，其性能提升相当亮眼。而本周早些时候的传闻，也提到过 3 月 21 日这个时间点。</p><p>此前我们已经知晓四款 EPYC Milan-X CPU SKU，包括：</p><blockquote><p>● EPYC 7773X -- 64 核 / 128 线程，OPN 代码 100-000000504 。</p><p>● EPYC 7573X -- 32 核 / 64 线程，OPN 代码 100-000000506 。</p><p>● EPYC 7473X -- 24 核 / 48 线程，OPN 代码 100-000000507 。</p><p>● EPYC 7373X -- 16 核 / 32 线程，OPN 代码 100-000000508 。</p></blockquote><p><img src="https://static.cnbetacdn.com/article/2022/0226/2518cdd766579ef.png" alt="3.png" referrerpolicy="no-referrer"></p><p style="text-align: center;">（图 via ShopBLT）</p><p>性能方面，<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://amd-cpu.jd.com/" target="_blank">AMD</a> 通过 RTL 测试表明 Milan-X 较 Milan CPU 提升 66%，且 16 核 Milan-X 能够较非 -X SKU 更快地完成 Synopsys VCS 验证测试。</p><p>延迟和带宽方面，Milan-X 霄龙处理器也被证明可在拥有 768MB 超大缓存（且价格差异微小）的情况下具有更大的吸引力。</p><div class="article-relation"><p><strong>相关文章:</strong></p><p><a href="https://www.cnbeta.com/articles/tech/1240991.htm" target="_blank">AMD霄龙Milan-X零售价曝光：两成溢价换来三倍缓存 性能增益抢眼</a></p><p><a href="https://www.cnbeta.com/articles/tech/1242389.htm" target="_blank">传锐龙R7-5800X3D、Milan-X霄龙、线程撕裂者PRO 5000新品即将到来</a></p></div>
