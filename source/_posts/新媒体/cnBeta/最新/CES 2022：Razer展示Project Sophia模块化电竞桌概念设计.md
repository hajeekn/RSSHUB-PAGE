<div>   
在携手 D-Box 推出 Enki Pro HyperSense 动感电竞椅的同时，<strong>Razer 还在 CES 2022 期间，展示了代号为 Project Sophia 的模块化电竞桌。</strong>尽管仍处于早期概念阶段，但 Razer 对于模块化计算机的愿景，却可一路追溯到 2014 年亮相过的 Project Christine 项目。<br>
 <p><img src="https://static.cnbetacdn.com/article/2022/0106/81370b556e1d0ab.jpg" alt="0.jpg" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://razer.jd.com/" target="_blank">RAZER</a> <a href="https://www.razer.com/concepts/razer-project-sophia" target="_self">官网</a>）</p><p>新披露的 Project Sophia 概念，旨在将传统电竞桌变身为一台模块化的 PC 组件，看起来有点类似于《星际迷航》中的 LCARS 控制台。</p><p style="text-align: center;"><iframe src="//tv.sohu.com/s/sohuplayer/iplay.html?bid=316405944&autoplay=false&disablePlaylist=true" width="640" height="480" frameborder="0"></iframe></p><p style="text-align: center;">Project Sophia - World's First Modular Gaming Desk Concept（<a href="https://tv.sohu.com/v/dXMvODIyMjQwNTMvMzE2NDA1OTQ0LnNodG1s.html" target="_self">via</a>）</p><p>2014 年的 Project Christine 项目，特点是采用了易交换的积木（砖头）塔式机箱设计。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/df910ebdd97616d.jpg" alt="1.jpg" referrerpolicy="no-referrer"></p><p>2020 年的 Tomahawk 项目，则借鉴了更切实际的<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://intel.jd.com/" target="_blank">英特尔</a> Element NUC 理念。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/bf2dd47a2906d7d.jpg" alt="2.jpg" referrerpolicy="no-referrer"></p><p>言归正传，如图所示，Razer 为 Project Sophia 新概念设计了 13 个不同的可交换模块插槽。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/7f2d2361d469dce.jpg" alt="3.jpg" referrerpolicy="no-referrer"></p><p>用户可自由添加不同的部分，包括温度读数、触屏应用启动器、聊天 / 日历显示、Qi 无线充电板、杯子加热垫、数位板、混音器、CPU / GPU 指示器等。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/a234ede59e46c5a.jpg" alt="4.jpg" referrerpolicy="no-referrer"></p><p>模块化的特性，很适合用户根据特定的需求和用例而展开调整，比如流媒体主播能够借此轻松连接扬声器、麦克风、网络摄像头等组件。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/946a150e46eee9b.jpg" alt="5.jpg" referrerpolicy="no-referrer"></p><p>额外的显示屏可用于查看与粉丝之间的聊天互动内容，而视频编辑器有助于添加混响效果，以提升工作的专注度。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/07b31d6d4a772c4.jpg" alt="7.jpg" referrerpolicy="no-referrer"></p><p>至于实际的“PC”部件，Project Sophia 概念也设想将之嵌入电竞桌中。定制的磁性连接机箱可轻松拆卸，以便用户后续更换 CPU / GPU 等部件。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/dc345d4518afc7b.jpg" alt="8.jpg" referrerpolicy="no-referrer"></p><p>整套设备可直面巨大的 <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https%3A%2F%2Flist.jd.com%2Flist.html%3Fcat%3D737%2C794%2C798%26ev%3D4155_88234%26sort%3Dsort_rank_asc%26trans%3D1%26JL%3D2_1_0%23J_crumbsBar" target="_blank">OLED</a> 面板（65 / 77 英寸显示器），辅以诸多“灯厂”特色的 RGB 灯效。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/91cc41d86642ef4.jpg" alt="9.jpg" referrerpolicy="no-referrer"></p><p>遗憾的是，Razer 似乎没有将 Project Sophia 模块化概念电竞桌快速推向市场的计划。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/fc97464f38e862a.jpg" alt="10.jpg" referrerpolicy="no-referrer"></p><p>此前的 Project Christine（2014）、Project Valerie 三屏笔记本（2017）、Project Linda <a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://shouji.jd.com/" target="_blank">手机</a>底座（2018）等项目，也都没能如愿以偿。</p><p><img src="https://static.cnbetacdn.com/article/2022/0106/5390bb8cc8d3e3e.jpg" alt="11.jpg" referrerpolicy="no-referrer"></p><p>毕竟与标准化的 PC 硬件组件相比，模块化产品的市场规模实在太过小众。</p><div
