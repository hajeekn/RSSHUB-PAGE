<strong>OpenAI 在周四上午的公告中表示，GPT-3 已经变得更加开放。</strong>随着这家人工智能研究机构取消访问其自然语言处理（NLP）应用程序接口（API）的等候名单，此举意味着 OpenAI 已对该程序感到足够安全和自信，后续将有助于推动业内领先的读写 AI 模型的访问。<br>
 <p><img src="https://static.cnbetacdn.com/article/2021/1119/c61af8a86b7b5f2.png" alt="0.png" referrerpolicy="no-referrer"></p><p>Axios 指出，来自受支持地图的开发者们，将很快能够注册访问 OpenAI 的 GPT-3 API 并立即开展体验。</p><p>此前，开发者需要先坐在等候名单上、静待官方审核通过，然后才能获得实验性质的访问权限。</p><p>OpenAI 产品与合作伙伴关系副总裁 Peter Welinder 表示：</p><blockquote><p>我们对 API 实施了大量改进，并引入了诸多安全特性。而能够让更多的开发者构建其问题解决方案的计划，有助于创造更大的价值。</p></blockquote><p>据悉，GPT-3 是一种基于文本的生成  AI 模型，训练期间被投喂了互联网上的 5000 亿词汇量，并涉及 1750 亿个参数。</p><p>OpenAI 希望借此调教出最佳的 AI 实践，当它在去年夏天发布的时候，也是当时最大、最复杂的自然语言处理（NLP）模型。</p><p>实测表明 GPT-3 能够根据提示而生成让人感到颇为震惊的拟人文本，并可分析总结庞大的非结构化书面语言数据源。</p><p><img src="https://static.cnbetacdn.com/article/2021/1119/e7fc026a9324654.gif" alt="1.gif" referrerpolicy="no-referrer"></p><p style="text-align: center;">（来自：OpenAI <a href="https://openai.com/blog/openai-api/" target="_self">官网</a>）</p><p>OpenAI 表示，数以万计的开发者通过其 API 访问了 GPT-3 模型，并构建了包括客户反馈回答、机器人客服、以及大文案输出在内的各种应用程序。</p><p>然而现阶段，AI 拟人文本技术的采用还是引发了一些争议，尤其担心其在虚假宣传等活动中被滥用 —— 即便 Welinder 宣称到目前为止的概率非常低。</p><p>作为应对，OpenAI 还更新了其社区准则，以屏蔽仇恨言论、试图影响政治进程的内容、以及除健康和性<a data-link="1" href="https://c.duomai.com/track.php?k=yUSQzUycwRHdo1Ddm0DZpVXZmkDN2ITPklWYmYDO5IDNy0DZp9VZ0l2cmYiJ05WZkVHdzZkMl42Yu02bj5SZy9GdzRnZvN3byNWat5yd3dnRyUiR" target="_blank">教育</a>之外的所有成人内容。</p><p>总而言之，OpenAI 希望从一个非盈利性的研究机构，转变成为一家与<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>达成合作的盈利性公司，且正在积极致力于拿到相关产品的商业许可。</p>
