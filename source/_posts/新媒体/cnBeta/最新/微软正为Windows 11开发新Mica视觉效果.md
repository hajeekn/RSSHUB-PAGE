援引科技媒体 Windows Latest 报道，微软正计划在 Windows 11 上开发测试“Tabbed” Mica 特效。在 Windows 11 Build 22523 的 SDK 中，有一个新的“Tabbed”设计公共 API。这是由 Start Is All Back 首先发现的，<strong>当在 Acrylic、Mica 和 Tabbed 之间切换时，你可以看到设计的变化。</strong><br>
 <p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/133efbaf91e4d11.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/0b5f9112607feef.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/d1eb936861b471b.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">在浅色模式下，切换到“Tabbed”选项就会启动 Mica 类玻璃效果，会有一个严重模糊的壁纸。<a data-link="1" href="https://c.duomai.com/track.php?site_id=242986&euid=&t=https://www.microsoftstore.com.cn/" target="_blank">微软</a>还提供了一个体验的样本应用程序，显示了这些 API 可能对一个真正的应用程序产生的影响。我们在下面为你收录了这些，尽管我们认为切换到黑暗模式时，它更明显。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/d4a1354bf877f90.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/3b1880dc3199ba0.webp" referrerpolicy="no-referrer"></p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2021/1228/b6a7523811f2d7f.webp" referrerpolicy="no-referrer"></p><p style="text-align: left;">这个外观相当有趣。它似乎很像 Mica，但又不像半透明。但它可能不是 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a> 11 的唯一变化。根据Windows Latest，微软可能愿意将 Mica 效果带到更多现代和传统的应用程序中。这显然可以从最新的 Windows Insider 构建中的“MicaBackdropInApplicationFrameHostTitlebar”标志看出。</p><p style="text-align: left;"><iframe src="https://cdn.knightlab.com/libs/juxtapose/latest/embed/index.html?uid=10a0744a-6777-11ec-abb7-b9a7ff2ee17c" width="720" height="320" frameborder="0"></iframe></p>
