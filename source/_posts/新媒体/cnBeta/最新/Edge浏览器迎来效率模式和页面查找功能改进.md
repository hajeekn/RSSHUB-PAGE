随着功能的持续完善，Microsoft Edge 在浏览器市场的份额也在稳步增长。对于部分用户来说，Microsoft Edge 的体验优化，还是较 Google Chrome 或 Mozilla Firefox 更具吸引力。<strong>而在最近的一次更新后，微软又为 Edge 浏览器带来了所谓的“效能模式”（Efficiency Mode）。</strong><br>
 <p><a href="https://static.cnbetacdn.com/article/2021/1120/c5fefc1a913a18d.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1120/c5fefc1a913a18d.jpg" alt="1.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: center;">（图 via <a href="https://www.windowslatest.com/2021/11/19/microsoft-edge-gets-efficiency-mode-and-improved-find-on-page/" target="_self">Windows Latest</a>）</p><p>此前，这项功能又被称作“性能模式”（Performance Mode），允许用户优化其 Web 浏览的响应能力。</p><p>不过随着更名为“效能模式”（Efficiency Mode），该模式可以更贴切地允许用户调教优化，以减少 CPU 和内存资源的使用，从而延长电池续航和改善系统体验。</p><p>当然，“效能模式”的实际体验，仍取决于用户设备、应用程序、浏览器使用习惯等因素。</p><p><a href="https://static.cnbetacdn.com/article/2021/1120/4fdce70dd96c1fb.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1120/4fdce70dd96c1fb.jpg" alt="2.jpg" referrerpolicy="no-referrer"></a></p><p>微软官方介绍称，基于 Chromium 内核的 Edge 浏览器，会在闲置 5 分钟后、让标签页转入后台休眠（当前仅在“标签页休眠”启用时生效）。</p><p>不过在使用该功能时，用户可能会遇上一些问题。比如在用户并未主动使用浏览器时，Microsoft Edge 的动画效果会变慢 —— 尤其在最小化浏览器、打开另一个应用程序、然后再返回当前页面时。</p><p>此外这项功能可能导致视频播放不太流畅，所以在收看 YouTube 或 Netflix 等流媒体节目时，还请手动禁用 Edge 95 中的该功能。</p><p><a href="https://static.cnbetacdn.com/article/2021/1120/64eb07143c38965.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/1120/64eb07143c38965.jpg" alt="3.jpg" referrerpolicy="no-referrer"></a></p><p>另一方面，微软改进了在 Microsoft Edge 上的“页面内查找”宫。当前在用户输入想要检索的关键词时，浏览器会弹出一个小搜索框，但又缺乏对相关匹配的支持。</p><p>好消息是，在迭代更快的 Edge Canary 通道中，微软正在测试新的“在页面上查找”工具。其带有新的切换开关，以启用包含相关匹配的选项（类似与 Microsoft Word 或记事本）。</p><p>最后，微软还在 Edge 中试验 <a data-link="1" href="https://microsoft.pvxt.net/P0JMe" target="_blank">Office</a> 集成，预计相关功能将于 2022 年初登陆稳定版通道。</p>
