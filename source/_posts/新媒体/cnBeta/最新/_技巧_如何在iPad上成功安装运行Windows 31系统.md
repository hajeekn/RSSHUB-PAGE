<strong>Fast Company 的科技编辑 Harry McCracken 近日找到了一个方法：成功在 iPad 上安装 Windows 3.1 系统。</strong>他使用了一款名为 iDOS 的应用程序，通过它可以在 iPad 上安装支持第三方应用程序的经典桌面环境。
How-to Geek 的 Benj Edwards 已经把如何使用 iDOS 在 iPad 上启动和运行 Windows 3.1 的<a href="https://www.howtogeek.com/739100/how-to-install-windows-31-on-an-ipad/" target="_blank">步骤指南</a>分享出来。<br>
 <p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0714/7c37f91b5c59a26.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0714/7c37f91b5c59a26.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">iDOS 是一个用于 iOS 和 <a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fipad%2F" target="_blank">iPad</a>OS 的应用程序，旨在让你在现代<a data-link="1" href="https://apple.pvxt.net/c/1251234/435400/7639?u=https%3A%2F%2Fwww.apple.com%2Fcn%2Fmusic%2F" target="_blank">苹果</a>设备上玩经典的 DOS 游戏。虽然它不一定要使用完整的 <a data-link="1" href="https://microsoft.pvxt.net/x9Vg1" target="_blank">Windows</a>，但这是可能的。它也恰好很容易让它启动和运行。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0714/cabb53a54b39cf4.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0714/cabb53a54b39cf4.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">想要运行 Windows 3.1，你需要以 4.99 美元的价格在 App Store 上购买 iDOS，你还需要一个合法的 Windows 3.1 副本。</p><p style="text-align: left;">一旦你得到了 Windows 3.1 的文件，你需要把它们转移到你的 iPad 的 Files 应用程序中。你要确保它最终出现在那里。然后，你可以在 iDOS 中加载 Windows 3.1 文件并安装操作系统。你应该遵循 Benj Edwards 的详细指南，以确保你正确配置一切。</p><p style="text-align:center"><a href="https://static.cnbetacdn.com/article/2021/0714/1cbc25b189312c5.jpg" target="_blank"><img src="https://static.cnbetacdn.com/thumb/article/2021/0714/1cbc25b189312c5.jpg" referrerpolicy="no-referrer"></a></p><p style="text-align: left;">事实证明，你实际上可以通过iDOS在Windows 3.1上安装额外的第三方应用程序。本杰明还介绍了如何利用互联网档案馆获得Windows 3.1的软件。你可以在那里找到游戏、生产力工具、实用程序等。通过iDOS和Windows 3.1，你可以把你的iPad变成一台便携式的经典游戏机。Benj展示了在他的iPad上运行的几个经典Windows游戏，包括《文明II》等游戏。</p>
