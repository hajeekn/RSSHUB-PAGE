<strong>Chrome 97 稳定版于今天正式发布。</strong>诚然，Chrome 97 在终端用户功能方面并不令人振奋，但新版本针对开发者补充了一些新功能。在 Chrome 97 版本中最值得关注的是以类 WebSockets 接口引入了 WebTransport API。<br>
 <p style="text-align: left;">WebTransport是一个类似于WebRTC数据通道的协议框架，但主要是用于受网络安全模型限制的客户端，使用安全、多复式传输与远程服务器进行通信。WebTransport 使用 HTTP/3 协议进行双向传输。与基于 TCP 的 WebSockets 不同，WebTransport 依赖于类似 UDP 的数据包和可取消的流。</p><p style="text-align:center"><img src="https://static.cnbetacdn.com/article/2022/0105/adb93370c414b6a.webp" alt="j7l2sm99.webp" referrerpolicy="no-referrer"></p><p style="text-align:
