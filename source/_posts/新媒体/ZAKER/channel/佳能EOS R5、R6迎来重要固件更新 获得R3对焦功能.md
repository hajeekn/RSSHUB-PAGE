<p>在近日，佳能推出了 EOS R5、EOS R6 的 1.5.0 版新固件，新固件在功能上得到大幅度提升，从 EOS R3 上下放对焦功能，能够识别汽车、摩托车。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202112/61a985148e9f0942c943f4a2_1024.jpg" data-height="500" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61a985148e9f0942c943f4a2_1024.jpg" referrerpolicy="no-referrer"></div></div>EOS R5、EOS R6 最新固件均为 1.5.0 版，它们改进点近乎一致，相同点有：<p></p><p>1. 头部检测性能已得到改进，以此为冬季运动过程中使用的头盔或护目镜提供支持。</p><p>2. [ 车辆 ] 选择已添加至 [ AF ] 选项卡中的 [ 检测的被摄体 ] 选择当中，以此为赛车运动中的汽车和摩托车检测提供支持。在将 [ 车辆 ] 选作驾驶员或乘员的盔甲时，将添加 [ 定点检测 ] 。</p><p>* 无法针对一般的乘用车或脚踏车进行检测，也无法针对表演特技或扬尘技艺的摩托车手进行检测。</p><p>3. 检测性能已针对眼部检测进行改进，例如：</p><p>- 面部存在阴影时。</p><p>- 头发遮住眼部时。</p><p>4. 人体躯干检测现已得到支持。如果眼部、面部或头部被遮挡或无法检测到，则将对人体躯干进行检测并持续不断进行追踪。</p><p>5. 在拍摄静止图像时，现在已经能够从 " 快速控制 " 屏幕导入手动白平衡 数据。</p><p>6. 为了防止 FTP 传输期间意外传输图像，已除去多控制器的中心按压操作功能。</p><p>7. 在同时插入两张存储卡，并使用 EOS Utility 进行远程拍摄时，现在可以设置记录方式并选择要记录的存储卡。</p><p>8. 已将 [ 抑制低帧频 ] 添加到 [ 显示性能 ] 的 [ 流畅 ] 当中。这可以降低所显示的拍摄屏幕的掉帧率，即使在黑暗的环境下也是如此。</p><p>9. RF 5.2mm F2.8 L 双鱼眼镜头现已得到支持。</p><p>10. [ FTP 服务器 ] → [ 被动模式 ] 的默认值已变更为 [ 启用 ] 。</p><p>11. 改善了 FTP 传输期间的操作稳定性。</p><p>12. 优化了在使用未配置图像稳定功能的特定 EF 镜头时，内置图像稳定性的控制。</p><p>从官方说明中可以看出，新固件主要是增强了自动识别功能，能够识别更多目标，或是提高识别能力，以及支持新推出的 RF 5.2mm F2.8 L。此外，EOS R5 还修复了在使用 Canon Log 进行影片录制时，明亮物体边缘出现噪点的问题。不过考虑佳能此次更新固件多次出现 bug，建议把 R5、R6 作为干活机的用户先观望一下。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202112/61a985148e9f0942c943f4a3_1024.jpg" data-height="500" data-width="750" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61a985148e9f0942c943f4a3_1024.jpg" referrerpolicy="no-referrer"></div></div>在同时，佳能还更新了旗舰单反相机 EOS-1D X Mark III 的 1.6.0 版固件，新固件改进如下：<p></p><p>2. 为了防止 FTP 传输期间意外传输图像，已除去多控制器的中心按压操作功能。</p><p>3. 在将 " 评分 " 功能分配给 " 保护 / 语音备忘录 " 按钮后，现已可以通过长按按钮录制语音备忘录。</p><p>4. 在同时插入两张存储卡，并使用 EOS Utility 或 Browser Remote 进行远程拍摄时，现在可以设置记录方式并选择要记录的存储卡。</p><p>5. [ 连接选项设置 ] 、 [ FTP 传输设置 ] 和 [ 节电 ] 的默认值已变更为 [ 关闭 ] 。</p><p>6. [ FTP 服务器 ] → [ 被动模式 ] 的默认值已变更为 [ 启用 ] 。</p><p>7. 改善了 FTP 传输期间的操作稳定性。</p><p>8. 修复了在使用 Canon Log 进行影片录制时，明亮物体边缘出现噪点的问题。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
