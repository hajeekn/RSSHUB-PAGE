<p>IT 之家 1 月 28 日消息，根据即将到来的软件更新的开发者发布说明，苹果将不再把 Python 2.7 与 macOS Monterey 12.3 系统捆绑。自 2020 年 1 月 1 日起，Python 2 不再被支持，并且不再收到任何错误修复、安全补丁或其他变化。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202201/61f4189a8e9f09062a7b2362_1024.jpg" data-height="810" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202201/61f4189a8e9f09062a7b2362_1024.jpg" referrerpolicy="no-referrer"></div></div>苹果公司表示，开发者应该使用替代的编程语言，如 Python 3，但值得注意的是，Python 3 也没有预装在 macOS 系统上。开发人员可以在 Terminal 终端运行 stub /usr/ bin / python3，但它提示用户需安装 Xcode 开发者工具，其中包括 Python 3。<p></p><p>IT 之家获悉，苹果公司早在 2019 年就警告说，在 macOS Catalina 的开发者发布说明中，未来版本的 macOS 将不包括 Python 2.7。当时，苹果表示，Python 2.7 包含在 macOS 中是为了与传统软件兼容。</p><p>" 如果你的软件依赖于脚本语言，建议你在应用中捆绑运行时。" 苹果说。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
