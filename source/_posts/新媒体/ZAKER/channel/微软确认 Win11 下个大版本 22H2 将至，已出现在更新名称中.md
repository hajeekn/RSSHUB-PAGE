<p>IT 之家 3 月 7 日消息，微软将在一周年之际发布代号为 "Sun Valley 2" 的 Win11 重大更新，该版本正在开发中，我们已经从 Dev 预览版了解到了部分新内容。</p><p>而在上周发布的 Windows 11 Build 22567.100 ( KB5012427）的 Dev 预览版中，微软明确提到了 "22H2" 这个字样，说明新版本已经不远了。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202203/6225afe48e9f097dda3bb844_1024.jpg" data-height="112" data-width="866" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/6225afe48e9f097dda3bb844_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，从目前的更新内容来看，22H2 版本将更侧重于改善 Windows 11 的现有界面，如开始菜单、控制中心 / 通知、MS Paint 等原生应用、Windows 设置、任务栏桌面和操作系统的其他区域。<p></p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202202/621456d08e9f093cee722619_1024.jpg" data-height="365" data-width="696" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/621456d08e9f093cee722619_1024.jpg" referrerpolicy="no-referrer"></div></div>此外，该更新还将重新支持任务栏拖放功能，并为 Windows 11 添加了新的任务管理器。<p></p><p>Windows 11 Build 22567.100 ( KB5012427）的 Dev 预览版并没有什么大的改动，因为它是一个累积性的更新，微软正在测试其服务管道。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202203/6225afe48e9f097dda3bb845_1024.jpg" data-height="900" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/6225afe48e9f097dda3bb845_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
