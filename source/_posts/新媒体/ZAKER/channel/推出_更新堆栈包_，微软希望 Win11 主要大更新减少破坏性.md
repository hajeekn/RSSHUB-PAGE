<p>IT 之家 2 月 2 日消息，在预览版本中，微软为 Windows 11 增加了新的 Windows 更新功能，称为 " 更新堆栈包 "（Update Stack Packages）。与功能体验包或功能更新不同，这些堆栈包包括一系列关键的改进和 shell 组件，使主要的 Windows 版本更新不那么具有破坏性。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202202/61f9d7728e9f09770441c38e_1024.jpg" data-height="753" data-width="1440" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/61f9d7728e9f09770441c38e_1024.jpg" referrerpolicy="no-referrer"></div></div>微软没有给出关于这些堆栈更新包如何工作的详细信息，但它可以使操作系统更加模块化。利用 " 更新堆栈包 " 和 " 功能体验包 " 等功能，微软可以在常规的操作系统更新之外为某些 Windows 模块推送新功能或改进。<p></p><p>Windows 更新堆栈包将在典型的预览版或安全版之外自动安装，而用户在检查更新时将注意到它的存在。今天，微软发布了一个新版本的堆栈包，版本为 822.124.211.0，它并没有带来新的功能。</p><p>该 Windows 堆栈包的目的还没有明确，但它可能被用来提供更新体验的改进。</p><p>这些软件包将在每月的累积更新或功能更新之前发布，微软此前表示，该功能将有助于确保你的 PC 保持稳定。</p><p>目前，" 更新堆栈包 " 仅限于一小部分与更新有关的改进，但你可以期待在未来的版本中进行更多的改进。目前，微软只想在主要的 Windows 版本之外测试与更新有关的小系统文件。</p><p>" 更新堆栈包 " 通过 Windows Update 交付，就像累积更新一样。换句话说，你会在设置 > Windows Update 中找到这些堆栈包。</p><p>除了 Windows 更新堆栈包和 Windows Update 的其他几个变化，Windows 11 的下一个功能更新也包含了其通常的新功能和改进。例如，" 设置 " 应用程序已经更新，为管理 Windows 更新提供了更多选项，你通常会在控制面板中找到。</p><p>Windows 更新的卸载按钮</p><p>现在，如果你想卸载更新，你需要浏览控制面板中的 Windows Update 页面，自 Windows 1.0 以来，这一直是 Windows 的一部分。未来随着 Windows 11 22H2（又称太阳谷 2）的推出，微软终于将 " 卸载 Windows Update" 功能迁移到了现代设置应用程序中。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202202/61f9d7728e9f09770441c38f_1024.jpg" data-height="784" data-width="1121" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/61f9d7728e9f09770441c38f_1024.jpg" referrerpolicy="no-referrer"></div></div>从截图可以看出，Windows 11 即将推出的 " 卸载更新 " 功能并不像控制面板那样全面，而且缺少过滤器或栏目等功能，但微软已经承诺在未来的版本中进行改进。<p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
