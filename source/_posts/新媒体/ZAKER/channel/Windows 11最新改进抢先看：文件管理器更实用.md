<p>对于不少关注 Windows 11 的用户来说，在最新的 Build 22563 版本中，微软已经偷偷上线了不少新功能，其中系统中的文件管理器变的更为实用。</p><p><strong>在 Windows 11 最新预览版中，在文件管理器中增加了使用右键菜单将某个文件固定的功能。而在此前，文件管理器只支持固定文件夹。一旦你固定住一个文件，你就能在快速访问中的 " 固定住的文件 " 部分看到它，如下面的截图所示：</strong></p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/6219a28a8e9f09361711aa73_1024.jpg" data-height="335" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/6219a28a8e9f09361711aa73_1024.jpg" referrerpolicy="no-referrer"></div></div>只是有点遗憾的是，固定的文件无法出现在左侧的快速访问侧边栏中，这有点令人沮丧。<p></p><p>Windows 11 Build 22557 版本中开始引入的一项新功能，是能够看到文件夹中项目的预览。当你将文件夹视图设置为除列表或详细信息之外的任何内容时，就会出现这种情况。上面可以看到一个带有超大图标的例子。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202202/6219a28a8e9f09361711aa74_1024.jpg" data-height="335" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/6219a28a8e9f09361711aa74_1024.jpg" referrerpolicy="no-referrer"></div></div>文件夹预览是以前所有版本的 Windows 中都有的一个方便功能，它是许多人每天都在使用的东西，所以它被 Windows 11 删除的事实激怒了许多用户。从 Windows 11 Build 22557 开始，现在可以在资源管理器中预览文件夹的文件，而无需打开该文件夹。<p></p><p>在昨天发布的 Windows 11 Build 22563 中，文件管理器还引入了其他一些后台改进。首先是快速访问搜索现在包括来自 OneDrive、下载和任何索引位置的内容。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202202/6219a28a8e9f09361711aa75_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/6219a28a8e9f09361711aa75_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
