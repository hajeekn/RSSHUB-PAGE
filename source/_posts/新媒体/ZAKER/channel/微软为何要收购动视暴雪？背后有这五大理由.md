<p>1 月 19 日消息，据外媒报道，软件巨头微软宣布斥资 687 亿美元、以全现金方式收购动视暴雪，将视频游戏领域的两股力量整合起来。这是微软有史以来规模最大的一笔收购交易，价值几乎是 2016 年收购职业社交网站 LinkedIn 的 3 倍。以下为微软做出收购决定的五个关键理由。</p><p><b>1、规模效应</b></p><p>如果能获得监管部门的批准，这笔交易将帮助缔造全球第三大游戏公司，使微软一跃成为仅次于腾讯和游戏机竞争对手索尼的游戏巨头。动视暴雪首席执行官鲍比 · 科蒂克 ( Bobby Kotick ) 在接受采访时承认，该公司无法独自在新的游戏世界中竞争。</p><p>科蒂克解释称：" 你可以看看 Facebook、谷歌、亚马逊和苹果这样的公司，尤其是腾讯，它们的规模都非常庞大，我们意识到需要一个合作伙伴才能实现自己的梦想和抱负。" 微软首席执行官萨蒂亚 · 纳德拉 ( Satya Nadella ) 周二对投资者表示，对于微软来说，" 我们共同的抱负是把游戏的欢乐带给地球上的每一个人 "。</p><p></p><div class="img_box" id="id_imagebox_0" referrerpolicy="no-referrer"></div></div><b>2、移动端</b><p></p><p>这是游戏增长最快的部分。动视暴雪拥有移动游戏工作室 King，后者开发了有史以来最受欢迎的手游之一《糖果粉碎传奇》 ( Candy Crush ) 。但微软在手游领域几乎没有任何影响力。</p><p>周二被任命为微软游戏 ( Microsoft Gaming ) 部门首席执行官的 Xbox 主管菲尔 · 斯宾塞 ( Phil Spencer ) 在接受采访时说：" 我们都知道，当今世界上最受欢迎的游戏设备是手机。" 与微软有类似动机的可能还有美国游戏开发商 Take-Two Interactive Software，该公司本月早些时候宣布斥资 110 亿美元收购了手机游戏开发商 Zynga。</p><p><b>3、绕过 " 苹果税 "</b></p><p>纳德拉希望他的游戏帝国足够庞大，玩家可以绕过苹果公司的应用商店直接进入这个帝国。微软始终在与苹果和谷歌就应用商店对游戏应用收取的应用内购买佣金方面存在争执。</p><p>纳德拉在与投资者举行的电话会议上表示：" 今天，我们面临着来自许多公司的激烈竞争，这些公司从游戏发行中获得的收入超过了我们从游戏销售和订阅中分得的份额。我们需要在内容创作方面加大创新和投资，减少对发行的限制。" 斯宾塞在谈到手游时也曾提到：" 目前移动设备上的分销由两个人、两家大公司（苹果和谷歌）控制，而微软想要自己‘不受约束地’发行游戏和内容的能力。"</p><p><b>4、元宇宙</b></p><p>游戏是微软进军 " 元宇宙 " 的两大法宝之一，另一个是 Office 和会议软件。纳德拉和斯宾塞已经将围绕《我的世界》（Minecraft）和《光环》（Halo）等游戏成长起来的游戏玩家社区视为类似于元宇宙的概念。</p><p>收购动视暴雪后，微软将能够提供更多规模更大、更专注的游戏社区来创建自己的元宇宙。纳德拉说：" 当我们思考自己的元宇宙可能会是什么样子的时候，我们相信不会也不应该有单一的集中式元宇宙。我们需要支持许多元宇宙平台，以及强大的内容、商业和应用生态系统。"</p><p><b>5、" 三 C" 战略</b></p><p>纳德拉的公司战略始终在围绕所谓的 " 三 C" 进行整合，它们包括云计算服务（cloud）、内容（content）和创作者（creator）。微软希望将动视暴雪过去和未来的内容尽可能多地放到其云游戏服务 Xbox Game Pass 上，后者一直通过稳定的订阅收入支撑着这家软件制造商的销售。</p><p>此外，这笔交易还允许微软鼓励更多玩家创建自己的游戏内容和虚拟世界。在加入了《我的世界》、LinkedIn 和 GitHub 之后，纳德拉始终在收购大量资产，这将使他获得更加庞大的创作者社区。尽管纳德拉未能领导微软收购短视频服务 TikTok 的美国业务，与 Pinterest 和 Discorde 的谈判也无果而终，但在收购动视暴雪的最新尝试中，他似乎取得了成功。（小小）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
