src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61f1dc05b15ec07a8d20ecbd_1024.jpg" referrerpolicy="no-referrer"></div></div>周三下午，游戏聊天与应用社区 Discord 突发了一场大范围崩溃，导致许多用户无法正常联网。中断开始于美国东部时间 14:45（中国时间凌晨 3：45） 左右，不仅语音通话无法连接，而且文本聊天也无法正常使用。<p></p><p>" 我们目前正在调查这次大规模 API 中断，并正在努力尽快解决这个问题。"Discord 官方发推说。</p><p>3 个小时后，Dicord 的问题似乎已经解决。其帐户在美国东部时间下午 17:19 （中国时间凌晨 6:19）发布了 " 现在应该已经完全恢复连接 " 的推文。</p><p>Discord 被游戏社区广泛使用，目前每月有 1.5 亿活跃用户。这次中断似乎对大多数用户都产生了影响，其连接中断的新闻在网络上也掀起了很大讨论。</p><p>Discord 在他们客户端的一条消息中说：</p><p>" 我们已经确定了 API 中断的根本问题，但正在处理一个数据库集群上的次要问题。我们有整个待命响应团队在线并紧急修复这次的故障。"</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202201/61f1dc05b15ec07a8d20ecbe_1024.jpg" data-height="371" data-width="740" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61f1dc05b15ec07a8d20ecbe_1024.jpg" referrerpolicy="no-referrer"></div></div>中断结束后，其状态页面上发布了一条消息，称 " 我们已完全取消所有速率限制，Discord 几乎恢复正常。" 它还指出，" 在接下来的一个小时内，一些 Discord 服务器可能会发现一些使用 " 斜杠 " 命令与机器人交互的问题。作为解决事件的一部分，我们需要减少数据库的负载，并且我们关闭了斜杠命令系统的某些部分。"<p></p><p>Discord 上一次出现短暂中断是在 11 月，那次是由于谷歌云出现问题，导致了通信应用的崩溃。而这次最近的停机时间在 Discord 的状态页面上被标记为 " 重大中断 "，足见这次崩溃影响了相当大量用户的正常使用。</p><p>过去几年，Discord 一直在悄悄建立自己的基于机器人的应用程序平台，并计划在春季发布新的应用程序发现功能。机器人是 Discord 体验的重要组成部分，超过 30% 的 Discord 服务器都在使用它们。从播放音乐到投骰子，Discord 的机器人能在用户聊天、交流的过程中起到十分有趣的作用。</p><p>但同时，机器人的使用也给 Discord 的运营带来了不少麻烦，其中一个重要的影响就来源于机器人对 Discord 数据库的压迫。为了处理机器人使用中产生的大量信息，Discord 推出了不少技术（如网关意图识别、提醒管理等）来保障数据库的正常运转。而这次崩溃，代表 Discord 在机器人的技术支持上，还有很长的路要走。</p><p>在 QQ 上，也曾经有过机器人的 " 大繁荣 " 时期。但在 2020 年，大量的机器人被腾讯封禁。当年 7 月，晨风机器人的作者称，由于腾讯起诉，自己被宿迁警方以 " 非法入侵计算机系统罪 " 传唤并拘留。此后，QQ 仍在持续对机器人采取禁用的措施。</p><p>时至今日，QQ 上已经鲜有机器人的踪迹。</p><div
