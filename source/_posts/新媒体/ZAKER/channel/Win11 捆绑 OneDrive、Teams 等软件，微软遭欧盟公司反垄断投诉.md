<p>IT 之家 11 月 27 日消息，今年 10 月，微软开始向符合条件的 Windows 10 个人电脑推出 Windows 11 正式版免费升级服务。</p><p>据 MSPoweruser 报道，由文件托管公司 Nextcloud 牵头的欧盟企业联盟向欧盟和德国反垄断机构提起诉讼，称微软滥用其在操作系统领域的主导地位，向协作和云服务等无关领域扩张。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/61a1888e8e9f097f5b26cf5f_1024.jpg" data-height="675" data-width="1200" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/61a1888e8e9f097f5b26cf5f_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，这一企业联盟表示，微软正在将 Microsoft 365 深入地集成到 Windows 在内的服务和软件组合中，并称每当 Windows 用户处理文件存储时，OneDrive 都可作为 Windows 操作中的一种选择。另一个例子则是 Teams 软件，它作为 Windows 11 的默认应用被成功推向市场。<p></p><p>他们声称，微软将占主导地位的 Windows 操作系统与 OneDrive 云产品相结合，几乎不可能与他们的 SaaS 服务竞争。这说明了基于 Windows 市场支配地位的 " 自我偏好 " 等反竞争行为。</p><p>在云存储领域，微软、谷歌和亚马逊的市场份额在过去几年已经增长到整个欧洲市场的 66%，而本地（欧洲）提供商的份额则从 26% 下降到 16%。</p><p>该企业联盟指出，上述行为是全球科技巨头在欧洲飞速增长的核心，与 90 年代末微软将 Internet Explorer 与 Windows 捆绑在一起，从而摧毁网景（Netscape）的情况非常相似。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
