<p>IT 之家 2 月 13 日消息，微软近期宣布推出 Windows 预览体验计划的全新图标。微软表示，新图标代表热爱并通过 Dev、Beta 和 Release 预览 3 个内部通道帮助塑造 Windows 未来的用户们。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202202/62086b30b15ec0758f18ff82_1024.jpg" data-height="434" data-width="752" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/62086b30b15ec0758f18ff82_1024.jpg" referrerpolicy="no-referrer"></div></div>IT 之家了解到，Dev 通道的 Insider 成员很快就会在 " 设置> Windows 更新 > Windows 预览体验计划 " 下看到该图标。新图标也将出现在电子邮件和其他区域中。<p></p><p>此前，微软还公布了最新的 Windows 预览体验计划变动。Dev 通道将成为实验性功能出现的地方，而 Beta 通道则包含更接近所有用户的功能。这可能意味着新功能如果接近公布的话，甚至会首先出现在 Beta 通道中。</p><p>根据微软的描述，微软还计划为 Windows 11 测试人员提供一个选项，可以从 Dev 通道切换到 Beta 通道，来避免更多正在进行的实验性功能。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
