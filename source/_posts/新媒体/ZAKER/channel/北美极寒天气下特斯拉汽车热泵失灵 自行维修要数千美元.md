<p>1 月 13 日消息，<strong>受极寒天气影响，最近一些特斯拉电动汽车的热泵系统严重失灵，特斯拉车主在驾驶过程中饱受严寒困扰。</strong>去年特斯拉 Model 3 和 Model Y 电动汽车就出现过热泵失灵的问题，随后特斯拉声称通过远程无线更新解决了这个问题，但今年冬季又卷土重来。</p><p>目前，加拿大和美国北部的气温创下了历史新低，一些地区的气温甚至低于零下 30 摄氏度。</p><p>很多特斯拉电动汽车车主称车内供暖系统出现和去年一样的问题。同样，似乎只有配备热泵系统的特斯拉 Model 3 和 Model Y 车型受到这个问题的困扰。</p><p>众所周知，热泵系统在极端寒冷的环境中 ( 约零下 15 摄氏度以下 ) 无法正常工作。</p><p>特斯拉声称已经用双循环系统解决了这个问题，但似乎还有其他问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202201/61dfebbb8e9f09187c57963f_1024.jpg" data-height="325" data-width="660" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202201/61dfebbb8e9f09187c57963f_1024.jpg" referrerpolicy="no-referrer"></div></div>但有几位特斯拉车主称，系统发出警报，车里完全没有热量。<p></p><p>在寒冷的冬季，这有可能造成潜在的危险情况。因为一些特斯拉车主或许会被困在温度极低、没有供暖的偏远地区。</p><p>问题发生后，几名把车开到特斯拉服务中心的特斯拉车主被告知，特斯拉已经发现这个问题，正在进行软件修复。</p><p>来自加拿大多伦多的 Jimmy Yeung 发现自己的 Model Y 发生同样问题。特斯拉服务中心为其更换了车载空调压缩机和热泵系统的 " 八通阀 " 部件。</p><p>Jimmy Yeung 在 Facebook 上分享了自己自掏腰包维修特斯拉汽车的账单。账单显示，如果特斯拉汽车不在保修期内，车主为此需要花费超过 4000 美元。</p><p>特斯拉也告知几名热泵出现问题的 Model 3 和 Model Y 车主，问题可能很简单，比如冰堵住了挡风玻璃附近的前进气口，导致热泵系统传感器出现错误。</p><p>公司建议使用预热功能来解冻或手动清除进气口的冰。</p><p>特斯拉首席执行官埃隆 · 马斯克（Elon Musk）此前曾称 Model 3 和 Model Y 电动汽车所搭载的热泵系统是 " 他这段时间见过的最好工程实例之一 "。</p><p>如果自己的 Model 3 或 Model Y 电动汽车安装了热泵系统，那么特斯拉车主在寒冷地区进行长途公路旅行时应该考虑到这一点，因为在这些地方没有供暖会带来真正危险。</p><p>特斯拉最好能很快找到根本解决方法，毕竟对于过保车辆来说，自掏腰包更换配件太过昂贵。</p><p>当然，人们也不用就此对电动汽车持否定态度。相关特斯拉汽车虽然不供暖，但至少还能开。</p><p>在极寒温度下，许多内燃机车都会有起动问题。此外，这一问题仅限于配备热泵的特斯拉汽车。没有安装热泵系统的特斯拉汽车运行正常。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
