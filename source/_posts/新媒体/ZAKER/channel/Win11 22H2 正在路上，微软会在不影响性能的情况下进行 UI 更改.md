<p>IT 之家 12 月 19 日消息，微软已经开始着手开发下一个 "Windows 11 22H2" 大版本，预计将于 2021 年秋季发布。</p><p>作为 Windows 11 22H2 版本的一部分，微软计划通过将 Mica+Fluent 引入更多使用不同框架的应用程序来改进现有操作系统的界面。</p><p>Fluent Design 是 Windows 11 和 Windows 10 的设计语言的一部分，它采用了光和景深等元素来取代 Windows 8 单调的外观。它拥有类似 Acrylic" 亚克力 " 这般的设计，这是一种用于菜单、对话框背景或整个应用程序窗口的半透明材质。</p><p>在 Win11 中，微软为其 Fluent 设计语言添加了一种新的材质，即 Mica" 云母 "。这也是一种半透明的模糊设计，可以让你的桌面壁纸在应用程序窗口中隐隐发光。在某种程度上，它也可以实现类似于 " 亚克力 " 或 Win7 的 Aero" 毛玻璃 " 效果。</p><p>云母的透明度效果可以在省电模式或者当设备速度过慢（硬件过老）时自动调整，所以它在保持亚克力材质的优势的同时实现了更快更流畅的效果。</p><p>IT 之家了解到，Win11 的 Mica 材质目前还无法与所有现代化应用程序兼容，但这种情况可能很快就会改变。</p><p>根据 WindowsLatest 发现的新参考内容，微软正计划将 Mica 扩展到现代应用程序甚至传统应用程序的 "FrameHostTitlebar" 标签。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 等应用中，例如标题栏。得益于此，用户无需从微软商店下载更新包就可以获得一些额外的 UI 优化。</p><p>新材质</p><p>有趣的是，微软似乎正在试验另一种名为 "Tabbed" 的设计材料，这种材料似乎是以云母为基础进行优化的产物，也可能是云母的一种版本。</p><p>在 Windows 11 Build 22523 SDK 的 Win32 API 中，用户发现了新的和奇怪的 " 标签 "
