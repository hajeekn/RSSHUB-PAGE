<p>苹果虽然在 2019 年推出了 Pro Display XDR 显示器，但那个起步价显然不是一般视频 up 主和摄影师所能接受的，直到今个春天，苹果才再一次面向 " 主流 " 级内容创作市场，推出了 Studio Display，分辨率达到 5K，并内置有苹果自家芯片，能够与生态内的其它设备完美使用，无论对于 Mac Studio ，还是 MacBook Pro 用户，都会是最佳搭配的显示器。</p><p>iFixit 目前已经在制作 Studio Display 的拆解内容，不过他们在打开 Studio Display 后，就被苹果这台高素质显示器的内部给震撼到了，迫不及待地先分享了一张照片，虽然在此之前，大家都知道了 Studio Display 内部相当于藏有一台 iPhone 11，因为苹果给它塞入了一颗 A13 bionic 芯片，用于进行相机的降噪、Center Stage 以及 Siri 语音识别等在内的 AI 处理功能，但没想到作为一台显示器，Studio Display 可以做得如此复杂。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202203/623af743b15ec01c376b97b9_1024.jpg" data-height="675" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/623af743b15ec01c376b97b9_1024.jpg" referrerpolicy="no-referrer"></div></div>照片来源 iFixit<p></p><p>虽然 iFixit 还没有分析讲解 Studio Display 内在各个部分的信息，但大家也不难看出，Studio Display 的内部结构和一台 iMac 很相像，主板的右边有一套完整的 A13 bionic 平台，以及接口和各类管理芯片，而左边有内置电源适配器，另外考虑到 5K 面板的发热量，还内置有两个主动散热风扇，当然要说夸张的，自然就是扬声器部分了，占据有机身左右很大面积，这也是为什么有些评测称 Studio Display 的外放，已经可以满足基本的监听需要了。</p><p>可以说 Studio Display 远超过一台显示器所需，硬件上是个超大号的 iPad 了，或者苹果原本是准备做 iMac 27 英寸（目前在售只有 iMac 24 英寸），只是后来遇到不明原因，才临时修改为显示器，当然这些只是我们的猜测而已，但不得不说，苹果这次也是刷新大家对于显示器内部的认识。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
