<p>IT 之家 2 月 4 日消息，微软已经对 Windows 功能体验包进行了近两年的测试，但它的真正目的却不为人知。如果你现在使用的是 Windows 11，那么你之前可能已经下载并安装了两个体验包 —— Windows 特性和 Web 体验包。</p><p>但目前微软还没有充分利用好 Windows Feature Experience Pack 的真正潜力。WindowsLatest 报道称，Windows 体验包将从 Windows Core OS 中分离出一些功能或主要功能，因此微软无需发布新版 Windows 11 极氪更新某些功能或引入对 Android 应用程序的支持。</p><p>到目前为止，Windows 体验包都是关于文件资源管理器、剪切工具和其他系统方面的小改进和 bug 修复。在一份新的声明中，微软确认 Windows 11 的 Insiders 频道将会更新，并提供新的实验性功能。</p><p>这些新功能将通过 Windows 功能、Web 和在线服务体验包 ( Online Service Experience Packs ) 在预览版构建之上发布。</p><p>微软表示 :" 我们将通过发布功能、Web 和在线服务体验包 ( Feature, Web, Online Service Experience Packs ) 来更新开发和测试渠道的功能和体验。"</p><p>微软将在 Windows 11 中启用一些实验性或隐藏的功能</p><p>有一些开发人员发现，微软在预览版本中有意禁用了一些特性。但微软现在希望为更多用户提供 beta 版功能，并分阶段试验某些功能，以监控质量和测试变化。</p><p>Windows 11 测试版功能</p><p>有很多功能可以通过体验包进行改进，例如，Windows Web 体验包的更新可以在 Windows 11 Widget 板中启用新功能。同样，微软也可以使用功能体验包 ( Feature Experience Packs ) 将更改推送到 Windows 设置中。微软将会测试更多用户的变化，以改善每个人的体验。</p><p>当然，没有人能保证开发渠道或体验包的变化最终会作为周二 Windows 补丁或功能更新的一部分面向所有人。IT 之家曾报道，微软还表示，正计划让 Windows 11 测试人员测试、评估更多的功能，<strong>但</strong>这些功能可能永远不会发布。</p><p>本月晚些时候，微软将开始向公众推出 Android 应用程序、任务栏和其他改进。目前，我们还没有任何关于微软计划如何启用这些功能的信息。</p><p>此外，从今年开始，Windows 11 将进入年度功能更新计划。除功能体验包外，你还可以期待微软在 Windows 11 秋季更新中带来一些令人兴奋的功能。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/61fd1ecc8e9f09337156dd2e_1024.jpg" data-height="461" data-width="820" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/61fd1ecc8e9f09337156dd2e_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
