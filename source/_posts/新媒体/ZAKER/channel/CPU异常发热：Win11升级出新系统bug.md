<p>芯研所 2 月 13 日消息，日前，微软面向 Beta 和 Release Preview 频道的 Windows Insider 用户推送了 Build 22000.526 ( KB5010414 ) 系统。</p><p>微软在更新日志中写道：" 如果任务栏在中心位置对齐，我们将天气内容添加到任务栏的左侧。当你将鼠标悬停在天气上时，Widgets 面板将出现在屏幕的左侧，当你停止悬停在该区域时将消失 "。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/6208d03d8e9f0942de58fa24_1024.jpg" data-height="361" data-width="640" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/6208d03d8e9f0942de58fa24_1024.jpg" referrerpolicy="no-referrer"></div></div><strong>芯研所采编</strong><p></p><p>然而 Win11 Build 22000.526 升级不可避免又带来了一波新 bug，目前主要集中在两个方面，一个是打印机无法工作，有网友表示卸载这次的更新之后恢复正常，显然是要这个升级补丁背锅。</p><p>另一个问题就是 CPU 异常高热，因为 Win11 Build 22000.526 系统会导致 CPU 处于高负载状态，导致温度不正常。目前微软还没有确认 Win11 Build 22000.526 系统的问题，还没升级的可以观望下。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
