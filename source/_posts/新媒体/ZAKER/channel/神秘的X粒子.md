神秘的罕见粒子<p></p><p>在大爆炸后的百万分之一秒内，宇宙是一团汹涌、混乱、高温的<strong>夸克和胶子等离子体</strong>。之后，随着宇宙的膨胀和冷却，夸克开始互相结合，形成一种名为<strong>强子</strong>的复合粒子。强子包括由一个夸克和一个反夸克组成的<strong>介子</strong>，以及由三个夸克结合在一起的<strong>重子</strong>，我们熟悉的<strong>质子和中子就都是重子</strong>。</p><p></p><div 图片设计：雯雯子<p></p><p>不过，在冷却前的混沌期，也就是在质子和中子形成之前，这些基本粒子会以数不尽的组合形式短暂地黏在一起。其中一部分夸克和胶子会在随机碰撞中，形成寿命短暂的<strong>"X" 粒子</strong>。正如它的名字那样，<strong>这些粒子有着神秘的、至今未知的结构</strong>。</p><p>现在，一个国际物理学家团队利用欧洲核子研究中心（CERN）的大型强子对撞机（LHC）所产生的夸克胶子等离子体数据，发现了一种 X 粒子存在的证据。他们将研究结果发表在了近期的《物理评论快报》上，标志着科学家<strong>首次在夸克胶子等离子体中探测到了 X 粒子</strong>。</p><p>X ( 3872 ) 的未知结构</p><p>中子和质子是普通物质的基本组成部分，它们都是由三个紧密结合的夸克组成的。在很长一段时间里，物理学家一直认为，出于某种原因，自然界中只有由两个或三个夸克形成的粒子。</p><p>然而，在上世纪 60 年代 , 物理学家<strong>默里 · 盖尔曼</strong>（Murray Gell-Mann）意识到，夸克或许还能以其他的方式组合。例如，两个夸克和两个反夸克或许可以黏在一起形成四夸克态，而四个夸克和一个反夸克可以黏在一起形成五夸克态。</p><p>近年来，物理学家开始发现奇异的四夸克态的踪迹。2003 年，日本 KEK 实验室的物理学家在进行 Belle 实验时，发现了一种名为<strong>X ( 3872 ) </strong>的新粒子。这种粒子是以其预估质量命名的，它们表现出了完全不同于普通介子的奇异特性。</p><p>Belle 实验是一项将高能的电子和正电子碰撞在一起的粒子对撞实验。在这种环境下，这些罕见的 X 粒子会迅速衰变，以至于科学家根本来不及详细研究它们的结构。因此时至今日，科学家仍不确切地知道 X ( 3872 ) 的内部结构，<strong>他们怀疑 X ( 3872 ) 要么是一种紧凑的四夸克，要么是一种全新的分子，这种分子不由原子组成，而是由两个松散结合的介子组成</strong>。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres2.myzaker.com/202201/61eeb837b15ec02c8c54a8fd_1024.jpg" data-height="638" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61eeb837b15ec02c8c54a8fd_1024.jpg" referrerpolicy="no-referrer"></div></div>四夸克态的两种可能构成。| 图片设计：雯雯子<p></p><p>有人提出，或许在夸克 - 胶子等离子体环境中，可以更好地揭示 X ( 3872 ) 和其他奇异粒子的奥秘。因为<strong>从理论上看，等离子体中所含有的大量夸克和胶子能够增强粒子的产生，但实际情况是，这样的夸克汤中还会有大量的其他粒子产生</strong>。因此要想通过这种方法来寻找 X 粒子，会是一项极为艰难的任务。</p><p>机器学习算法筛查海量数据</p><p>在新研究中，研究团队试图通过搜寻 LHC 的<strong>重离子碰撞所产生的夸克胶子等离子体</strong>数据来寻找 X 粒子的迹象。他们分析的是 LHC 在 2018 年生成的数据集，这些数据覆盖了<strong>130 多亿次的铅离子碰撞</strong>，每次碰撞都会释放出夸克和胶子。在冷却和衰变之前，这些夸克和胶子会散射和合并，形成超过<strong>10¹ </strong>个寿命短暂的粒子。</p><p>在夸克 - 胶子等离子体形成并冷却后，每次这样的碰撞都会产生大量粒子，这意味着在这样超密集的高能粒子汤中寻找 X 粒子时，所需面对的背景噪声是压倒性的。研究人员必须设法击破这个问题，才有可能最终在数据中看到 X 粒子。</p><p>为此，他们使用了一种<strong>机器学习算法</strong>，试图通过训练算法来识别 X 粒子的衰变模式特征。</p><p>粒子在夸克 - 胶子等离子体中形成之后，会立即分解成 " 子 " 粒子，然后散开。在这些粒子中，X 粒子的衰变模式，或者说角分布，有别于所有其他粒子。研究人员确定了描述 X 粒子衰变模式形状的关键变量，然后训练机器学习算法来识别这些变量，再将 LHC 碰撞实验产生的实际数据输入算法中。这种算法能够对极其密集和嘈杂的数据集进行筛查，挑选出可能代表了 X 粒子衰变的关键变量。</p><p>最终，研究人员成功地将背景噪声降低了几个数量级，从而看到了信号。他们将信号放大，<strong>在一个特定的质量位置观察到了一个峰值，表明了 X ( 3872 ) 粒子的存在，且其数量总共约 100 个</strong>。</p><p>故事才刚刚开始</p><p>从如此庞大的数据集中梳理出 100 个粒子，几乎是不可想象的挑战。研究人员对他们的结果进行了多次检查，以确保他们真的观测到了这样的信号。现在，他们已经成功地<strong>证实了 X 粒子可以在夸克胶子等离子体中被探测到</strong>。</p><p>不过，物理学家表示，这将只是故事的开始。虽然已经证明他们可以这样来寻找 X 粒子，但在接下来的几年里，他们希望利用夸克 - 胶子等离子体，能够帮助他们更详细地探测 X 粒子，以确定 X 粒子的内部结构。</p><p>X 粒子究竟是紧密结合的四夸克，还是松散结合的分子？目前，已有的数据对这两种可能性的支持是等同的，只有继续收集更多的数据才有可能帮助区分这两种情况。到那时，我们对产生于早期宇宙中的大量产粒子或许将有新的认知。</p><p><strong></strong></p><div id="recommend_bottom"></div><div
