<p>此前，微软为 Windows Server 2022 系统推送了编号为 KB5007205 的更新补丁，该补丁将修复此前出现在一些高分辨率屏幕上的缩放错误问题。</p><p>但近日，有消息称，该补丁在 Windows Server 2022 上造成了一个较为严重的故障，该故障可能会影响到系统的网络安全。</p><p>据悉，在此次更新后，有用户发觉 Windows Server 2022 上，只要安装有 Windows Server Core，那么终结点安全平台（Microsoft Defender for Endpoint）就可能出现无法运行的情况。</p><p>经了解，微软的终结点安全平台功能能够帮助企业用户进行网络预防、监测、调查以及响应高级别的威胁，对于 Windows Server 2022 这一主要针对企业用户的服务器系统来说极为重要。</p><p>Windows Server 是微软最早于 2003 年开始推出的 Windows 的服务器操作系统，而 Windows Server 2022 则是该系统最新的长期服务版本。</p><p>好消息是，微软已经确认了这一 Bug，并回应称正在进行解决方案的制定，在之后发布的新版本补丁中，将正式修复这个问题。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202111/61a4356ab15ec05ec717332c_1024.jpg" data-height="337" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202111/61a4356ab15ec05ec717332c_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
