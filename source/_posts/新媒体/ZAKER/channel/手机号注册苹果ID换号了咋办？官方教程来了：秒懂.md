<p>对苹果用户来说，想要使用 iOS 完整的服务就必须注册 Apple ID，如下载 App，使用 iMessage、FaceTime、iCloud 等服务。</p><p>那么如果用手机号注册完 Apple ID 后，手机号换号了怎么办？今日，@Apple 支持 官微对 Apple ID 常被问到的几个问题进行了回答。</p><p>官方表示，在 iPhone 或 iPad 上就能更改号码，首先退出登录所有 Apple 设备，只剩一台用来操作的设备除外。</p><p>随后前往 " 设置 "" 你的姓名 " 点击 " 姓名、电话号码、电子邮件 " 后，在联络方式选项选择编辑，删除现有号码，按提示添加新手机号，更改成功后，使用新账号重新登录所有服务即可。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/621445ad8e9f091f6e3fe25c_1024.jpg" data-height="732" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/621445ad8e9f091f6e3fe25c_1024.jpg" referrerpolicy="no-referrer"></div></div>另外，对于熟人之间借用 Apple ID 的问题，官方提醒，Apple ID 包含用户的通讯录、iCloud 照片、手机备份等个人资料，即使借个信赖的人，也相当于授权他访问帐号的所有内容，存在误编辑或误删某些数据的风险。<p></p><p>值得一提的是，有用户在评论区吐槽称：" 所以说登录验证怎么关闭，我有两个 Apple ID 每次登陆都需要验证码，真的烦死了，双重验证关都没法关 "。</p><p>对此，@Apple 支持 表示，" 针对此情况，若你的 Apple ID 升级为双重认证已超过两周，且未在注册后两周内通过邮件取消，则无法关闭双重认证。"</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202202/621445ad8e9f091f6e3fe25d_1024.jpg" data-height="5108" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/621445ad8e9f091f6e3fe25d_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
