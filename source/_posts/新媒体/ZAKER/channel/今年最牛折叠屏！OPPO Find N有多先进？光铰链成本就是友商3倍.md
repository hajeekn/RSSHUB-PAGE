<p>OPPO 首款折叠屏手机 OPPO Find N 已于不久前官宣，将于 12 月 15 日正式发布。</p><p>OPPO 首席产品官刘作虎透露，OPPO 突破性地解决了折痕、耐用性等折叠屏行业难题。想要突破这些行业难题，关键在于折叠屏手机的铰链，那么 OPPO Find N 的铰链有什么来头呢？</p><p>12 月 10 日，OPPO 官方针对 OPPO Find N 的铰链进行预热。OPPO 称，OPPO Find N 采用了全球首款精工拟椎式铰链，比 U 型铰链技术更加先进。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202112/61b4cca58e9f090e101e6165_1024.jpg" data-height="375" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202112/61b4cca58e9f090e101e6165_1024.jpg" referrerpolicy="no-referrer"></div></div>从视频来看，OPPO Find N 这款铰链采用水滴形设计，这种类型的铰链往往可以最大程度减少折痕，并且使用寿命也更长，但结构设计也更复杂，成本更高，据称该铰链单个成本在 800 元左右，是其他铰链成本的三倍。<p></p><p>值得一提的是，为 OPPO 提供精工拟椎式铰链的供应商是杭州安费诺飞凤通信，这家公司同时也是苹果的供应商，具备雄厚的技术，是全球最大的连接件公司。</p><p>在精工拟椎式铰链的坚持下，OPPO Find N 的折叠屏被隐藏得很好，照片显示 OPPO Find N 在展开状态下，没有明显的折痕，屏幕表面也非常平整，这种效果已经超越市面上大部分折叠屏了。</p><p>目前市场上在售的折叠屏手机存在三大主要问题，分别是折痕、软件适配和售价。</p><p>OPPO Find N 对折痕的处理显然已经到位了，那么在其他两个方面呢？</p><p>据了解，OPPO Find N 将搭载专门定制的 ColorOS 12，为折叠屏进行优化，官方号称 " 折叠交互新突破 "。</p><p>从官方发布的海报来看，OPPO Find N 能够实现类似 NDS 的游戏模式，下方的屏幕为操控按键，上半部分显示游戏画面，这或许将会是一种全新的折叠屏手机交互方式。</p><p>此外，OPPO Find N 搭载骁龙 888 处理器，支持任意角度悬停，X 轴线性马达，堪称是今年最牛折叠屏手机。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres1.myzaker.com/202112/61b4cca58e9f090e101e6166_1024.jpg" data-height="473" data-width="600" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202112/61b4cca58e9f090e101e6166_1024.jpg" referrerpolicy="no-referrer"></div></div><p></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
