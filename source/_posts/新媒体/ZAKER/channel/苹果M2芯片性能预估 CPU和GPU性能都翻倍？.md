<p>预计在 2022 年晚些时候，苹果将推出一款 "M2" 苹果芯片。</p><p>近日 MacWorld 的一位作者相信，通过研究苹果的 A 系列芯片，我们有可能更清楚地了解 "M2" 将在多大程度上进行升级。这份报告首先假设 M2 芯片将类似于 M1，就像 A15 仿生芯片与 A14 仿生芯片一样。</p><p>在此基础上，他推断出基于类似的提升，"M2" 的升级幅度到底会有多大。他写道："M1 基于 A14 的基本架构，将高性能 CPU 内核的数量增加了一倍 ( 从两个增加到四个 ) ，将 GPU 内核增加了一倍 ( 从四个增加到八个 ) 。"</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres1.myzaker.com/202202/6210ee3c8e9f0930b61e27dc_1024.jpg" data-height="608" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202202/6210ee3c8e9f0930b61e27dc_1024.jpg" referrerpolicy="no-referrer"></div></div>基本上，争论的焦点是下一代芯片将拥有两倍的高性能 CPU 内核和两倍的 GPU 内核。<p></p><p>这位作者继续说：" 在试图预测 M2 的功能和性能时，假设它将遵循类似的模式，只是这一次是 A15：相同的架构，CPU 内核和 GPU 内核提升两倍。"</p><p>然后，MacWorld 以 A14 和 A15 为基础，推算出新的 "M2" 能给 Mac 带来多大的性能提升。根据跑分估计，这可能是一个重大的跃升。</p><p>正如这篇文章指出的那样，"M2 Max" 的得分超过 14000 分，将远远领先于英特尔最新的 Alder Lake Core i9。当然，"M2 Max" 也可能会更省电。此外，推算的数据还显示图形性能显著提高，包括 3DMark 的 10409 分，高于 M1 的 7781 分。</p><p>当然，这些估计只是粗略的预测。"M2" 上的性能是否与推算的数据类似还有待观察，因为架构并不相同，预期的使用场景也不一样。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
