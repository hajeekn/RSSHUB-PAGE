<p>IT 之家 3 月 5 日消息，微软今天面向 Dev 频道发布了新的 Windows 11 Build 22567.100 ( KB5012427）的预览版更新。新的构建版本并没有带来任何新的内容，因为它是一个累积性的更新，微软正在测试其服务管道。</p><p>完整的公告如下：</p><p>" 我们开始推出累积更新 Build 22567.100（KB5012427）。这个更新不包括任何新的东西，旨在测试我们的服务管道在开发频道的构建。"</p><p>因此，早先发布的 Windows 11 Build 22567 的所有错误修复和改进都被带到了开发频道。该版本有大量的新功能，预示着正进入 Windows 11 的下一个阶段，为 22H2 版本做准备。</p><p>有趣的是，虽然在 Build 22567.100 预览构建中没有新的变化，但之前像这样的服务测试版本也破坏一些功能。例如，Windows 10 Build 21292.1010 对 ARM PC 上的 x64 模拟器造成了问题。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
