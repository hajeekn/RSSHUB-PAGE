<p>作为一个全新的操作系统，Windows 11 正式发布之后，少不了各种 BUG，比如硬件性能退步的问题，直到现在这一问题终于得到了彻底解决。</p><p>今天微软针对 Windows 11 用户推出了 KB5010386 累积更新。本次更新规模相当大，除了通过 Windows Update 常规方式进行升级之外，用户（尤其是企业管理员）可以通过 Microsoft Update Catalog 下载离线包，以便于快速部署。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202202/6203854a8e9f090923139766_1024.jpg" data-height="647" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202202/6203854a8e9f090923139766_1024.jpg" referrerpolicy="no-referrer"></div></div>KB5010386 累积更新除了安全更新之后，还整合了上月末的可选更新内容。其中修复了可能导致拖慢设备运行的错误。该错误虽然据称在 1 月的累积更新中得到了修复，但依然有部分用户抱怨在安装之后依然存在磁盘读写速度变慢的情况。<p></p><p>Windows 11 KB5010386 ( Build 22000.493 ) 重要更新日志：</p><p>修复了许多安全问题，包括一个使轻量级目录访问协议（LDAP）操作崩溃的问题。</p><p>引入了对帮助功能的支持，该功能基于微软 Bing 技术，它被用来推荐与每个 Windows 设置页面相关的帮助主题。</p><p>修复了一个恼人的错误，即操作系统为连接的蓝牙设备显示过时的电池百分比。</p><p>一些图像编辑程序在某些高动态范围显示器上无法正确呈现颜色的问题已经得到修复。</p><p>微软修复了一个影响 Windows 11 任务栏的自动隐藏功能的问题。</p><p><strong>· END<strong>·</strong></strong></p><div id="recommend_bottom"></div><div id="article_bottom"></div>
