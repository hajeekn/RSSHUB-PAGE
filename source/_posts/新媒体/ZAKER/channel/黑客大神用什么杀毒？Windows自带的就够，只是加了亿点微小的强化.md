<p>这年头 Windows 电脑还需要杀毒软件吗？</p><p>现在搜索这个问题，会发现很多建议都是 "<strong>裸奔就行</strong>"。</p><p>也就是说对于普通人，只用微软出厂自带的免费<strong>Windows Defender</strong>足够了。</p><p>那么非普通人，比如网络安全工程师、黑客们自己用什么？</p><p>没想到，他们也说 Windows 自带的就够了，只是需要<strong>再强化一下</strong>。</p><p>一位之前在公司里做网络安全主管，现在自己单干漏洞猎人的黑客大神 h0ek 抛出这个观点，在 Hacker News 上引起热议。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202203/622c31e98e9f094b60484fe0_1024.jpg" data-height="106" data-width="962" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/622c31e98e9f094b60484fe0_1024.jpg" referrerpolicy="no-referrer"></div></div>大神所说的强化也不复杂，就是开启了微软本来给企业客户准备的隐藏功能。<p></p><p>以前他们公司用的方案就是 Windows Defender 配合微软 Azure 上的云原生防护服务 Sentinel。</p><p>他发现一些高级功能其实在个人版和家庭版操作系统中也能开启，让个人电脑也拥有企业级防护。</p><p>开启隐藏功能</p><p>h0ek 所说的隐藏功能，全称为微软高级保护服务<strong>MAPS</strong>。</p><p>这是一种云安全防护，由知识图谱和机器学习驱动，用于识别病毒库里没出现过的全新威胁。</p><p>要启用这个功能需要用到组策略编辑器。</p><p>在 Windows 10/11 的专业版和企业版上，<strong>按下 Win+R →输入 "gpedit.msc"</strong>即可。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202203/622c31e98e9f094b60484fe1_1024.jpg" data-height="237" data-width="413" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/622c31e98e9f094b60484fe1_1024.jpg" referrerpolicy="no-referrer"></div></div>在组策略编辑器中找到<strong>计算机配置模版→管理模版→ Windows 组件→ Windows Defender 防病毒→ MAPS</strong>。<p></p><p>进入条目并选择启用。下面的选项可以选基本或高级，选择基本只会上传少量必要的数据。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_2" data-original="http://zkres1.myzaker.com/202203/622c31e98e9f094b60484fe2_1024.jpg" data-height="958" data-width="1452" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622c31e98e9f094b60484fe2_1024.jpg" referrerpolicy="no-referrer"></div></div>在个人版和家庭版上，组策略编辑器默认是用不了的，需要用脚本开启 MAPS 功能。<p></p><p>以管理员权限打开 PowerShell, 输入 "<strong>Get-MpRreference</strong>" 查看安全设置。</p><p>其中 MAPSReporting 控制 MAPS 的设置，0 是关闭，1 是基本模式，2 是高级模式。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_3" data-original="http://zkres1.myzaker.com/202203/622c31e98e9f094b60484fe3_1024.jpg" data-height="1450" data-width="1722" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622c31e98e9f094b60484fe3_1024.jpg" referrerpolicy="no-referrer"></div></div>输入 "<strong>Set-MpPreference -MAPSReporting 2</strong>" 就可以开启高级模式<p></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" 控制签名更新的间隔，h0ek 推荐设置成每小时 1 次。<p></p><p>CloudBlockLevel 控制云端阻拦级别，可以设置成 0，1，2，4，6，最高为 6，会阻拦所有未知的可执行程序。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202203/622c31e98e9f094b60484fe5_1024.jpg" referrerpolicy="no-referrer"></div></div>下一步是开启勒索软件专项防护。<p></p><p>这个就更简单了，直接在 Windows 安全控制面板里就有。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div"><img class="lazy opacity_0 zaker_gif_cache" id="img_6" 为荣，还有人觉得免费软件都是垃圾，这都没必要。<p></p><p>在他看来，Windows Defender 现在比最早刚推出时候进步了不少，加上强化以后，对普通用户来说足够了。</p><p>有杀软也要带脑子上网</p><p>有一位微软安全中心的员工看到这个分享，也跑来自夸一波。</p><p>作为工程师，我自己也承认这个产品真的很好。</p><p>他还爆料，产品的进步来自微软这些年并购了很多小型安全公司。比如最开始他负责的项目只有自己一个人在干，现在团队在全球有数百名成员。</p><p>这下可好，逮着一个内部员工，吸引了更多人来提问和提意见。</p><p>比如 Nim 编程语言的开发者就来求助。</p><p>因为使用 Nim 的黑客太多了，现在 Windows Defender 直接把 Nim 的安装程序都误报成了病毒，提交了误报核查目前还没有结果。</p><p>也有 Web 开发者来吐槽，说 Windows Defender 的安全防护能力确实够用，就是硬盘读写性能太差。</p><p>尤其是用 npm 安装的包文件太多，这是他还使用付费杀毒软件的唯一原因了。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_7"
