<p>前几天才说 Intel 正在准备 Alder Lake-HX 系列处理器，现在就在 UserBenchmark 的数据库里面找到一个记录，出现了一款名为 Core i7-12650HX 的处理器，虽然软件检测到的数据不完整，但依然可从中得知这款处理器部分规格。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202203/6221ead2b15ec074ad2731b2_1024.jpg" data-height="608" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/6221ead2b15ec074ad2731b2_1024.jpg" referrerpolicy="no-referrer"></div></div>这个 Core i7-12650HX 软件识别出来是 14 核 20 线程，而 Core i7-12650H 同样也是 14 核 20 线程，是由 6 个 P-Core 加 8 个 E-Core 组成的，然而 Alder Lake-HX 系列用的是桌面 Alder Lake-S 的核心，所以 Core i7-12650HX 较大概率是 8 个 P-Core 加 4 个 E-Core，也就是说和 Core i7-12700 类似。<p></p><p>Core i7-12650HX 的基础频率是 2.3GHz，平均睿频频率是 4.05GHz，但软件应该是把 P-Core 和 E-Core 的频率混到一齐算的，所以这平均频率没啥意义。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_1" data-original="http://zkres2.myzaker.com/202203/6221ead2b15ec074ad2731b3_1024.jpg" data-height="421" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/6221ead2b15ec074ad2731b3_1024.jpg" referrerpolicy="no-referrer"></div></div>核显肯定就是 UHD 770 那个级别的，这个测试平台其他的配置是一个三星 970 EVO Plus SSD，和 32GB+8GB 的 DDR5-4800 内存，这组合相当怪。<p></p><p>Alder Lake-HX 的基础功耗是 55W，比现在 Alder Lake-H 的基础功耗是 45W，最大功耗能到 115W，预期 Alder Lake-HX 的最大功耗肯定会比 115W 高，可能会到 150W，因此要用它的笔记本电脑肯定需要配一个效率相当高的散热器才行，此前泄露的产品线路图，Alder Lake-HX 至少会有一个 Core i9 和一个 Core i7 的型号，但 Core i7-12650HX 的出现说明 Core i7 型号至少会有两个，应该会有 Core i7-12700HX 这种产品的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
