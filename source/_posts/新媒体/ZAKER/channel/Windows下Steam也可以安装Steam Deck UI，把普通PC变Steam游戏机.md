<p>作为 Valve 的首款 PC 掌机，Steam Deck 的软件核心部分自然就是 Steam 客户端了，但与大家非常熟悉的客户端版有很大不同，它的 UI 界面有特别针对小屏幕和手柄操作，那硬件现在还买不到，软件总可以体验一下了吧，最近有玩家发掘到，其实 Windows 版的 Steam 也是可以使用这部掌机的 UI 界面。</p><p>在 Reddit 上有网友分享了如何把 Windows 版 Steam 客户端转变为 Steam Deck UI， 方法非常简单，先到 Steam 安装目录里面的 package 文件夹里面，创建一个名字为 beta 的 txt 文件，里面写上 steampal_stable_9a24a2bf68596b860cb6710d9ea307a76c29a04d，保存后把这个文档的 txt 后缀去掉，然后再到 Steam 的快捷方式的属性 - 目标里面，在文件路径的后面加上 -gamepadui，之后启动 Steam 就是 Steam Deck UI 了。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" data-original="http://zkres2.myzaker.com/202203/6225d8f48e9f097bee2fa98a_1024.jpg" data-height="607" data-width="1080" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202203/6225d8f48e9f097bee2fa98a_1024.jpg" referrerpolicy="no-referrer"></div></div>截图来自 ETA PRIME<p></p><p>由于 Steam Deck UI 针对手柄控制做了特别优化，所以对于一些把电脑作为 HTPC，或者本身是个 Steam 游戏狂，开机只玩 Steam 库里面游戏的话，那这个 UI 就很方便使用了，虽然 Steam 本来就有个大屏幕模式，但那个已经是多年前设计的了，可以说一点都不好用，相比之下，Steam Deck UI 就广受评测的好评，而且想回到传统的 Steam 界面也非常简单，只需要在快捷方式的启动项里删掉 -gamepadui 就可以了。</p><p>不过这也表明了其实在 Windows 系统下面，Steam Deck 掌机也是可以实现全手柄操作，但应该是考虑到系统授权费的问题，Valve 才选择使用 Linux 来作为底层，再搭配 Proton 兼容层，这样绕个圈地运行 Windows 平台下的游戏，不过目前在 Steam Deck 掌机上安装 Windows 还是很简单的，只是官方还没有完整的驱动支持，但之后玩家如果想用 Windows 11，但又要保留 Steam Deck UI 的话，应该是可以两者兼得的。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
