<p>2 月 26 日消息，美国当地时间周五晚间，电动汽车制造商特斯拉宣布，将把 " 纯视觉 " 系统引入 Model S 轿车和 Model X SUV 上，并承诺其司机辅助驾驶系统 Autopilot 和 " 全自动驾驶 "（FSD）套件将采用基于摄像头的感知方式。</p><p>早在 2021 年 5 月，特斯拉就宣布，从当月开始交付的所有 Model 3 和 Model Y 汽车将不再配备雷达。取而代之的是，这些车辆将利用特斯拉的纯视觉系统，这种系统只使用 8 个外部摄像头来支持半自动驾驶功能。</p><p>特斯拉在去年的博客文章中写道：" 从 2021 年 5 月开始，为北美市场制造的 Model 3 和 Model Y 将进行特斯拉视觉（Tesla Vision）系统优化，不再配备雷达。如果你在 5 月份之前下单，有可能会收到没有安装雷达的汽车，在交付之前，你的特斯拉账户会收到变动通知。"</p><p>特斯拉没有宣布何时将纯视觉系统引入 Model S 和 Model X，其正专注于首先在 Model 3 和 Model Y 上移除雷达，因为这两款车型的销量比 Model S 和 Model X 高得多，这使得特斯拉可以在短时间内获得大量数据。特斯拉表示，该公司将尽快确定将纯视觉系统引入 Model S 和 Model X 的合适时间。</p><p>周五晚间，特斯拉在其 " 过渡到特斯拉视觉 " 的网页上正式宣布，从 2 月中旬开始生产的所有 Model S 和 Model X 汽车都将只支持纯视觉系统，不再使用任何雷达来执行其功能。该公司写道：" 截至 2022 年 2 月中旬，所有为北美市场制造的 Model S 和 Model X 都使用特斯拉视觉系统，而之前为北美市场制造的所有 Model S 和 Model X 都配备了雷达。"</p><p>特斯拉表示，使用纯视觉系统的车辆可能无法使用某些功能。目前，Autosteer 的最高时速将被限制在 128 公里，跟随前车的距离也将比往常更长。特斯拉写道：" 在接下来的几周里，我们将开始通过一系列空中软件更新恢复这些功能。所有其他可用的 Autopilot 和 FSD 功能也将在交付时处于活动状态，这取决于订单配置。"</p><p>此举最终敲定了特斯拉向基于纯视觉方案转变的战略，而这也是马斯克长期以来一直决心要实现的目标。在 2021 年第一季度财报电话会议上，马斯克表示，汽车没有必要使用雷达，因为人类也只是使用视觉驾驶。根据马斯克的说法，车辆也只需依靠视觉。</p><p>马斯克说：" 当你的视觉正常工作时，它就像有 8 个摄像头，就像你的后脑勺、脑袋左右两侧也长了眼睛，并且有三只不同焦距的眼睛往前看。这是在以超人的速度处理数据。在我看来，毫无疑问，使用纯视觉解决方案，我们可以制造出比普通人驾驶安全得多的汽车。"</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
