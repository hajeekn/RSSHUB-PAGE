leitech</p><p>掐指一算，独立显卡短缺的问题已经持续两年时间了。在疫情导致的需求增加、缺芯带来的供给短缺以及挖矿潮带来的抢购风波下，各家厂商的独立显卡价格一直居高不下，就连一度被认为是游戏玩家救星的 LHR（锁算力）显卡也存在较高的溢价，让不少想更新换代的游戏玩家望而却步。</p><p>对玩家而言，如今的独立显卡就像空气一样，你能感觉到它的存在，但是看不见、摸不着。为了解决这个问题，在 2022 年国际消费电子展 ( CES 2022 ) 上，英伟达和 AMD 两家显卡厂商都推出了价格实惠的入门级游戏显卡。对于玩家来说，现在最重要的问题似乎只有一个，那就是这批显卡的国内起步价究竟会是什么水平？</p><p>近日，有网友发现，国内电商平台淘宝、闲鱼均有多家店铺上架了 Radeon RX6500XT 的预购链接，预购价格从 1699 元 -1799 元不等。从上架产品的店铺数量来看，这次 RX6500XT 显卡的铺货数量应该是十分充足的，而且产品的溢价总体来看也算是比较合理。</p><p></p><div class="img_box" id="id_imagebox_1" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " 独立显卡，也是目前市面上最有可能以接近原价的价格购买新品的入门级游戏显卡，RX6500XT 显卡是否值得普通玩家购买？这款显卡能否取代 GeForce GTX 1650/Radeon RX 580 的地位，成为新一代的廉价 GPU 王者？<p></p><p>新鲜出炉的 " 小鲜肉 "</p><p>先让我们过一遍参数，RX6500XT 显卡基于 Navi24 核心打造，采用台积电 6nm 制程工艺打造，内建 16 组 CU（共计 1024 个流处理器）和光追单元，配备 4GB GDDR6 显存，拥有 16MB 无限缓存，采用单 6Pin 外接供电，功耗 107W，游戏加速频率最高可达 2.6GHz，单精度 5.77 TFLOPS，支持 DP1.4+HDMI2.1 双输出。</p><p>与上一代同价位 GPU 相比，我们希望看到的肯定是更高的性能和更高的规格，而 RX6500XT 显然并非如此。从规格上看，RX6500XT 这张显卡只有 RX5500XT 一半的总线宽度、一半的 VRAM，和过往同价位 GPU 几乎一致的单精度浮点性能，但是价格反而更加昂贵。</p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img 帧，运行 APEX 平均可达 91 帧。作为对比，GTX 1650 在相同设置下运行使命召唤先锋平均可达 54 帧，运行 APEX 平均可达 71 帧；GTX 1660 在相同设置下运行使命召唤先锋平均可达 67 帧，运行 APEX 平均可达 112 帧。<p></p><p>换言之，在 AMD 官方设定的最理想的数据条件下，RX6500XT 就是一张性能介乎于 GTX 1650 和 GTX 1660 之间的入门级显卡。在游玩老游戏的情况下，RX6500XT 的性能表现和上一代 AMD 同价位 GPU 应该相差不大，在游玩新游戏的情况下，得益于 Radeon Super Resolution 和 Fidelity FX 两项技术支持，游玩帧率会更加理想。</p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_4" data-original="http://zkres2.myzaker.com/202201/61e61b648e9f091237418550_1024.jpg" data-height="350" data-width="592" src="https://cors.zfour.workers.dev/?http://zkres2.myzaker.com/202201/61e61b648e9f091237418550_1024.jpg" referrerpolicy="no-referrer"></div></div>问题来了，既然这张显卡性能并不突出，那么它到底有什么特点能够吸引玩家购买呢？答案可能让人意想不到，引用外媒 PCWorld 此前删除的访谈稿件，根据 AMD Radeon 副总裁劳拉 · 史密斯在面向外媒的通气会中的说法，RX6500XT 是一张在加密货币挖矿方面特地进行 " 优化 " 的显卡。<p></p><p>从过往的经验来看，阉割显存及位宽确实是打击矿工的关键，只有 4GB 显存、65bit 位宽的 RX6500XT 显卡，对于至少需要超过 4GB 的显存并且非常需要内存接口宽度的以太坊挖矿尤其不利，为了弥补较慢的内存接口带来的性能虽好，AMD 将 RX6500XT 的时钟频率提升许多，这也使得该卡挖矿功耗成倍提升。</p><p>目前看来，在没有矿工疯抢的情况下，RX6500XT 的预售价格整体是比较合理的。作为对比，目前市面上的二手 GTX 1650 售价在 1400 元左右，二手 GTX 1660 售价在 2000 元左右，二手 RX5500XT 8G 售价在 2200 元左右。考虑到 RTX 3050 的起售价要贵 50 美元，RX6500XT 有望成为市面上最廉价的全新入门级游戏显卡。</p><p>四处挨刀的 " 入门卡 "</p><p>俗话说得好，任何事物都有两面性。熟悉显卡厂商的消费者应该都很清楚，入门级显卡相对中高端显卡的唯一优势就是价格，其他的核心数、核心设计、显存大小、显存位宽、技术支持等参数基本上都是全面阉割的。问题在于，这次 RX 6500 XT 在阉割上感觉做得有点过了。</p><p>首先，不管 AMD 再怎么说，RX6500XT 始终都是一款只有 4GB 显存的独立显卡。显存是决定显卡性能的一个重要参数，如今游戏对硬件性能的要求很高，诸如《幽灵行动：断点》、《毁灭战士：永恒》、《生化危机 3》和《使命召唤：现代战争》重制版等游戏，在 1080P/ 高画质设置下显存要求轻松超过 4GB。一旦显卡爆显存，那么游戏帧率就会呈直线下滑。</p><p>其次，根据华擎 RX6500XT 产品页面显示的参数，我们可以知道这张显卡的 PCIe 总线带宽再次减半，现在只有 x4 通道，最高支持 PCIe 4.0x4 通道，该接口最高可以在 CPU 和 GPU 之间提供大约 8GB/s 的带宽。</p><p>在这种情况下，如果用户使用的是支持 PCIe 4.0 的最新主板，那么 PCIe 4.0x4 通道的带宽基本等效 PCIe 3.0 x8 通道，对于 RX6500XT 这样一款入门级显卡来说应该没有太多影响。如果用户使用的是仅支持 PCIe 3.0 的旧版主板，那就只有 PCIe 3.0x4 通道，这将对 GPU 和 CPU 之间的带宽造成相当严重的限制，进而限制 GPU 从内存中提取内容的能力，一定程度的性能下降是不可避免的。</p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202201/61e61b648e9f091237418551_1024.jpg" referrerpolicy="no-referrer"></div></div>最后，RX6500XT 显卡最大的问题是缺乏视频编码器 / 解码器，所谓视频编码器 / 解码器就像专门的小处理器，在制作 / 观看这些视频方面非常快速和高效。作为对比，其他 RDNA 2 GPU 都支持 H.264/AVC 和 H.265/HEVC 格式视频编码，然而 RX6500XT 显然完全没有这些功能，这款产品甚至无法正常解码 AV1 视频。<p></p><p>对于原本 CPU 强劲的用户而言，视频编码器 / 解码器的缺失可能不是什么问题，毕竟他们的 CPU 本身已经提供了足够强劲的编码 / 解码性能。问题在于，考虑到 RX6500XT 可能是这一代游戏玩家可用的最便宜的独立显卡，这张显卡显然会被预算更紧的消费者搭配较低端的 CPU 使用，这样视频编码器 / 解码器缺失带来的缺陷自然也被无限放大。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres1.myzaker.com/202201/61e61b648e9f091237418552_1024.jpg" data-height="448" data-width="985" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202201/61e61b648e9f091237418552_1024.jpg" referrerpolicy="no-referrer"></div></div>RX6500XT 是否值得购买？<p></p><p>问题来了，RX6500XT 是否值得购买？个人认为，在目前的市场环境下，想要买到一张不溢价的全新显卡显然不是易事，而 RX6500XT 则是解决这个问题的一个好办法。根据目前市面上的消息，RX6500XT 铺货非常充足，显卡原价出售只是时间问题，只要你愿意接受画质上的一定损失，这款显卡算是一个尚可的入门解决方案。</p><p>当然，如果你只是想玩游戏，并不在意全新显卡 / 二手显卡的区别，那么你的选择就多很多了。RX6500XT 羸弱的性能表现，让上一代显卡 GTX 1660/RX5500XT 看起来相当物超所值。如果你的预算还能略微提升一些，那么售价在 3000 元左右的 RX6600XT 也是一个不错的选择。</p><p>不得不说，RX6500XT 的存在更像是对如今 GPU 市场现状的一种控诉。这样一张性能平平、亮点乏善可陈的入门级显卡，居然因为 " 有货 "、" 不溢价 "、这种本应该显得稀疏平常的原因拥有了性价比。羸弱的性能，却因为不会被矿工哄抢而变成了优点，多少显得有些讽刺。</p><p>作为消费电子先锋展会的 CES 2021，本应该给我们带来性能焕然一新的 " 甜品 " 级显卡，而不是这样两款性能几乎没有提升的入门级产品。感谢 RX6500XT 能够以令人沮丧的性能配置为我们开启这本来就不抱什么希望的一年，在 " 缺芯潮 " 和愈演愈烈的矿潮之下，高端显卡的短缺可能还会持续很久很久。</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
