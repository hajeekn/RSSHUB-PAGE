<p>3 月 17 日消息，苹果在 2022 年度春季新品发布会上推出的 M1 Ultra 芯片对用户来说是好事，但对英特尔、AMD 和英伟达来说却是个大问题，这款苹果顶级自研芯片可能会颠覆计算机行业。</p><p>苹果刚刚发布了新款 M1 Ultra 芯片，售价 3999 美元、本周五开始发售的新款 Mac Studio 搭载的就是这款芯片。M1 Ultra 芯片性能可以媲美英特尔、AMD 和英伟达等诸多竞争对手的最好芯片产品。</p><p>这对于那些希望用新游戏平台和高端系统来运行视频和照片编辑软件的用户来说是个好消息。对于苹果来说，这也是一项巨大成就，因为苹果从 2020 年才开始为旗下笔记本电脑和台式机提供自家芯片。</p><p>对于那些想要一台能够完美处理各种任务的巨型计算机的消费者来说，这也意味着可以节省大量成本。苹果公司表示，在视频编辑领域，一款配备英特尔处理器芯片和 AMD 显卡的 Mac Pro 售价高达 25599 美元，而一款配备 M1 Ultra 芯片的 Mac Studio 价格只有 7999 美元。</p><p>7999 美元属实很贵，但这样一个软硬件系统要比搭载其他厂商处理器的 Mac Pro 便宜得多。韦德布什分析师丹 · 艾夫斯（Dan Ives）表示：" 苹果正在用 M1 系列芯片击败芯片制造商。"</p><p>当然，对苹果有利的事势必对其竞争对手不利。专家称，M1 Ultra 的横空出世可能会让消费者抛弃基于 Windows 系统的个人电脑，转而支持苹果 Mac 电脑。</p><p>市场研究公司 Gartner 高级总监级分析师乔恩 · 埃伦森（Jon Erensen）表示，" 苹果拥有自家芯片业务。它的功能非常强大，在应用处理器设计方面处于领先地位。"</p><p><b>苹果芯片为消费者提供了 Windows 之外的选择</b></p><p>多年来，一台能够处理大量数据的高端计算机往往需要搭载功能强大的 CPU 和独立显卡。Windows 电脑就是如此。</p><p>当然，苹果当时出售的也是搭载英特尔芯片和 AMD 显卡的高端电脑系统，但并不像其他个人电脑制造商那样频繁进行更新换代。很多个人电脑制造商每年都通过处理器的增量升级来推出新电脑，这使得 Windows 电脑成为市场上最强首选。</p><p>不过，苹果能凭借 M1 Ultra 芯片控制自家电脑系统中的 CPU 和 GPU，这意味着其可以根据需要更新 Mac 电脑。此外，M1 Ultra 芯片也可以更好集成苹果自家软件，有助于提升整体性能。</p><p>苹果使用一种名为 UltraFusion 的全新架构技术，将两个 M1 Max 芯片组合在一起，从而创造出顶级自研芯片 M1 Ultra。</p><p>M1 Ultra 芯片总共有 20 个 CPU 内核和 64 个 GPU 内核。据苹果公司称，这意味着 M1 Ultra 应对诸如 8K 高清视频编辑以及运行 3D 建模应用程序等高强度工作毫不费力。</p><p>苹果公司表示，搭载 M1 Ultra 芯片的 Mac Studio 将比搭载英特尔酷睿 i9 处理器和英伟达 RTX 3090 独立显卡的 Windows 电脑表现更好，同时整体能耗更低。</p><p>这不仅意味着 Mac Studio 运行时的温度比竞争对手的个人电脑更低，而且在峰值性能下持续运行的时间也更长。</p><p>由于产品设计和性能出众，苹果在创意设计等专业人士中一直很受欢迎。但近年来，随着微软等制造商推出搭载强大处理器和独立显卡的电脑系统，基于 Windows 系统的个人电脑开始在相关市场赶上苹果。</p><p>但凭借 M1 Ultra，苹果拥有了一款可以超越竞争对手的芯片，使苹果电脑的性能遥遥领先，这足以让诸多游戏玩家和开发者站到苹果这边。</p><p><b>苹果正在向英特尔、英伟达和 AMD 施加压力</b></p><p>目前 M1 Ultra 只用在 Mac Studio 电脑中。用户无法单独购买 M1 Ultra 芯片，然后把它塞进任何一台旧电脑里。这让那些喜欢自行组装电脑的极客很失望。</p><p>然而艾夫斯表示，苹果最终可能向其他电脑制造商提供 M1 Ultra 芯片，让消费者有能力自行组装基于 M1 Ultra 的电脑系统，这会让苹果与英特尔、AMD 和英伟达展开直接竞争。</p><p>艾夫斯表示：" 最新款 M1 Ultra 是显卡市场的颠覆者，最终将与英伟达展开竞争。"" 现在的问题是，苹果如何走出去，把自己的芯片卖给第三方。"</p><p>当然，苹果也可以简单保留自家芯片，并利用它们的优势来吸引更多客户。毕竟 iPhone 多年来都是如此。上世纪 90 年代，苹果曾将 Mac OS 操作系统授权给第三方，但后来取消了这一做法，因为苹果发现此举反而影响到自家 Mac 电脑的销售业绩。</p><p>如果苹果真的开始向第三方出售顶级芯片，可能会从英特尔、英伟达和 AMD 的市场份额中分得一杯羹。如果 M1 Ultra 性能像苹果公司声称的那样好，那么这一切发生的速度可能比想象的要更快。（辰辰）</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
