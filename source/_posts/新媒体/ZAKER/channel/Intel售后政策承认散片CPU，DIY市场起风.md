<p>尽管如今已是智能手机的时代，但是对于绝大多数朋友来说，手机还只是一个通讯工具或娱乐设备，距离生产力工具或高性能计算还有很长的一段路要走。事实上，随着此次疫情的全球流行，个人电脑（下文简称为 PC）也再次成为了消费电子领域备受追捧的对象。而除了购买联想、惠普、戴尔等品牌商推出的产品外，得益于 PC 高度模块化的特质，DIY 一台属于自己的 PC 同样也是很多人的首选。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_0" 的第一步。众所周知，如今在市场上关于 CPU 一般有盒装与散片两种产品，而两者之间的差别除了价格外，就要数保修政策了。不过就在不久前，Intel 方面在国内市场修改了其处理器保修政策，不仅在官方层面承认了此前处于灰色地带的散片，并且从 12 代酷睿开始，散装台式机处理器的有限保修期限同样也延长至 3 年。<p></p><p></p><div class="img_box" CPU，这其中制造工艺之复杂甚至难以用三言两语来概述，以至于在 x86 架构下的桌面级 CPU 市场也只有 Intel 和 AMD 两家才能玩转。<p></p><p></p><div class="img_box" id="id_imagebox_2" onclick><div class="content_img_div perview_img_div"><img class="lazy CPU 和散片 CPU 的差别就在于，盒装 CPU 有着精美的包装盒，并且除部分高端型号外一律提供原装散热器，而散片 CPU 顾名思义，就是只有一个简单的防静电泡沫 / 塑封的 CPU 本身，所以外观上的不同也是两者名字的来源。<p></p><p>更为重要的一点，是盒装与散片在商品属性上也完全不同。盒装 CPU 是作为单独的零售商品存在，有着属于自己的产品序列号，其销售是由 Intel 的市场部门负责，而散片则属于 PC 元器件，是由 Intel 的大客户部门负责，对接的是类似联想、惠普等 OEM 厂商。</p><p></p><div class="img_box" id="id_imagebox_3" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " PC 的 OEM 厂商，在向 Intel 订货的时候通常也都会预留出一定的冗余，以防止意外情况的出现，而散片的主要来源往往就是 OEM 厂商旗下产品出货量不及预期，或是主动释放库存的产物。<p></p><p></p><div class="img_box" id="id_imagebox_4" onclick><div class="content_img_div 自然也就对其不负有任何的义务。更何况，按照正常情况来说，Intel 会制止这种扰乱市场的行为，但因为其依赖于 OEM 厂商来销售自家的产品，以及 DIY 市场相对小众，所以在通常情况下 CPU 厂商对这种情况只是睁一只眼闭一只眼。<p></p><p></p><div class="img_box" id="id_imagebox_5" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_5" data-original="http://zkres1.myzaker.com/202112/61ae11b48e9f095c303e81f4_1024.jpg" data-height="600" data-width="573" src="https://cors.zfour.workers.dev/?http://zkres1.myzaker.com/202112/61ae11b48e9f095c303e81f4_1024.jpg" referrerpolicy="no-referrer"></div></div>而散片与盒装 CPU 之间的价格差异，除了 DIY 玩家通常不会看中的包装盒，以及散热效能相对普通的原装散热器外，拉开两者差价差的最大因素在于售后服务。事实上，CPU 作为极为精密的电子元器件，并不存在磨损通常也很难出现过热的情况，基本是 PC 中最为耐用的一个部分，但凡事都有例外，正因 CPU 非常耐用以及其精密的结构，所以也导致 CPU 一旦出现故障几乎就失去了维修的价值。<p></p><p>所以 Intel 对于 CPU 的官方保修政策，通常都是直接换新。而对于散片来说，由于 OEM 厂商不可能亲自在电商平台或线下亲自卖散片，所以其主要来源就是五花八门的各种网店或电脑城的线下店，通常主流的保修是提供一年 " 店保 "，也就是你从哪里买的，坏了就找他，再由其去找代理商，然后才是代理商一级一级往上申请即可。</p><p></p><div class="img_box" id="id_imagebox_6" onclick><div class="content_img_div perview_img_div"><img class="lazy opacity_0 " id="img_6" data-original="http://zkres1.myzaker.com/202112/61ae11b48e9f095c303e81f5_1024.jpg" data-height="381" 的故障判断非常麻烦，店方完全可以说是因为暴力超频或其他个人原因造成的人为损坏，而拒不认账。相较之下，盒装 CPU 的保修据网络中的用户反馈，除非是有明显人为损坏导致外观发生改变，Intel 一般都直接换新。<p></p><p>因此对于消费者来说，由于 Intel 此前并不认可散片的市场地位，而消费者购买散片其实是源自于对于 CPU 良品率高、且经久耐用特质的信任，带有一定 " 赌 " 的特色，所以大多数人购买散片时往往都不会对所谓的 " 店保 " 寄予厚望。而如今 Intel 在官方层面上承认散片、并且明确保修政策，这就等于是解除了这部分消费者的后顾之忧。</p><p></p><div class="img_box" id="id_imagebox_7" onclick><div class="content_img_div perview_img_div"><img 架构，使得其锐龙系列 CPU 的市场竞争力扶摇直上，从 Intel 手中抢下了一定的市场份额。而 AMD 的卷土重来多少让 Intel 感受到了压力，因此通过政策的利好来让更高性价比的散片与 AMD 的盒装 CPU 进行竞争。<p></p><p>不过对于想要体验 Intel 采用大小核混合架构设计 12 代酷睿的消费者来说，此次 Intel 对于散片保修政策的变化，想必也会让你再无后顾之忧了。</p><p>【本文图片来自网络】</p><div id="recommend_bottom"></div><div id="article_bottom"></div>
