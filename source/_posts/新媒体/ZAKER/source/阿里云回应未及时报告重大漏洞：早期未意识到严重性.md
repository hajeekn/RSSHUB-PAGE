<p>阿里云在官方微信公众号发布关于开源社区 Apache log4j2 漏洞情况的说明。说明称，阿里云因在早期未意识到该漏洞的严重性，未及时共享漏洞信息。</p><p>阿里云表示，Log4j2 是开源社区阿帕奇 ( Apache ) 旗下的开源日志组件，被全世界企业和组织广泛应用于各种业务系统开发。</p><p>阿里云称，近日，阿里云一名研发工程师发现 Log4j2 组件的一个安全 bug，遂按业界惯例以邮件方式向软件开发方 Apache 开源社区报告这一问题请求帮助。Apache 开源社区确认这是一个安全漏洞，并向全球发布修复补丁。随后，该漏洞被外界证实为一个全球性的重大漏洞。</p><p>阿里云表示，因在早期未意识到该漏洞的严重性，未及时共享漏洞信息。阿里云将强化漏洞管理、提升合规意识，积极协同各方做好网络安全风险防范工作。 ( 中新经纬 APP ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
