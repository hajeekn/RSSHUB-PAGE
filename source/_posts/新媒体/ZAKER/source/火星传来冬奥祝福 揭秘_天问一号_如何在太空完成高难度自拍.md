target="_blank">中新社</a>记者 郑莹莹</p><p>正在北京举行的冬奥会上，各国选手们比拼正酣。在遥远的太空中，" 天问一号 " 到达火星快一年了，它一边勤勤恳恳地探测火星，一边不忘携早已安装好的冬奥会会徽等元素，给地球发来 " 太空版 " 冬奥祝福。</p><p></p><div class="img_box" id="id_imagebox_0" onclick><div 供图<p></p><p><strong>我们有 " 宇宙级冬奥祝福 "</strong></p><p>北京是全球首个举办夏季、冬季奥运会的 " 双奥之城 "，" 天问一号 " 是中国首个行星际探索任务。这些年，伴随中国崛起，" 中国造 " 航天器相继奔太空，除了 " 天问一号 "，中国空间站也在如火如荼地建造中。</p><p>而早在 " 天问一号 " 奔赴太空之前，工作人员早早地就将北京冬奥会和冬残奥会吉祥物 ( 冰墩墩、雪容融 ) 和会徽的标志安装在它的身上。</p><p>此前，冰墩墩、雪容融 ( 的标志 ) 搭乘 " 天问一号 " 着陆火星，与祝融号火星车一起同框合影，梦幻联动。</p><p>新近，北京冬奥会开幕之际，一则穿越 3.2 亿千米的祝福又 " 从天而降 "，天问一号探测器、五星红旗与北京冬奥会和冬残奥会会徽又一次同框入镜。</p><p><strong>" 自拍达人 " 迎接太空拍摄挑战</strong></p><p>尽管 " 天问一号 " 近来也算是一个 " 自拍达人 "，从国庆、新年，到虎年新春，每逢大事件，它常自拍 " 美照 " 传回地球，以表思念与祝福，但其实要在太空中 " 好好自拍 "，是一件非常具有挑战性的事。</p><p>中国航天科技集团八院 803 所产品主管设计师吴迪介绍，在火星轨道上拍照的主要难点在于：被拍摄物体目标特性的不确定。日常人们在拍照的时候会根据光照条件选择一个合适的角度及拍摄模式，达到最佳拍摄效果。而火星轨道的天问一号环绕器 ( 简称：环绕器 ) 处于飞行过程中，其光照条件、受照范围、" 自拍杆 " 伸展后的角度都会导致国旗、冬奥会会徽以及环绕器本体等光学特性发生连续变化，而这些变化产生的不确定性，是对光学相机自动拍照的极大挑战。</p><p>另外，由于采用了视场更大的镜头，环绕器 " 入镜 " 面积大，可能出现部分处于极亮 ( 受太阳光照射 ) 状态、部分处于极暗 ( 阴影区域 ) 状态的情况，而国旗与冬奥会会徽的面积相对较小，随着环绕器在轨道上快速飞行，环绕器本体亮和暗的区域也一直处于变化中。</p><p>考虑到诸多拍摄工况，中国航天科技集团八院 803 所研制团队设计了以国旗为中心的自适应曝光成像控制策略。" 我们通过对获取图像情况的计算，自主调整曝光时间，在光照条件连续变化情况下，确保可以获取清晰的国旗图像。" 吴迪说。</p><p>从 " 天问一号 " 在北京冬奥会开幕之际传回的视频中，人们也可以看到，金色环绕器反射太阳光的位置在逐渐变化，从右下角逐渐向中间移动，直至强烈的阳光照射在中国国旗上，国旗始终清晰可见。 ( 完 ) </p><div id="recommend_bottom"></div><div id="article_bottom"></div>
