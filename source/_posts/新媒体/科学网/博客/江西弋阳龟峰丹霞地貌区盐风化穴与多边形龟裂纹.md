color: blue; font-size: 20px;">        </span><strong><span style="font-family: 宋体, SimSun; font-size: 20px; color: blue;">中国的江南和华南有大片的侏罗—白垩纪红层出露区，经过雨水冲刷和河流切割而形成以赤壁丹崖为特征的丹霞地貌。丹霞地貌形成的物质基础是红层（砂砾岩），而红层随着地壳隆升而出露地表，其断裂则成为水流切割的优选地带，因此沿着某些断裂形成陡崖。宏观上丹霞地貌的主要成因机制地学界已经弄清：降水沿着断裂渗透和河流切割而形成赤壁丹崖。然而，地学界对崖壁上分布的形态各异的洞穴却一直没有给出令人信服的机制解。几年前，我对广东北部的丹霞山世界地质公园、江西鹰潭的龙虎山世界地质公园进行了初步地质考察，明确指出这些洞穴是盐风化作用的结果，先后写成两篇博文发布于我在科学网的博客：</span></strong> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><span style="font-family: "Times New Roman"; color: blue;">        </span><strong><span style="font-family: 宋体, SimSun; color: blue;">遗憾的是，我当时竟漏掉了一个非常重要的景区：龙虎山世界地质公园龟峰景区。从网上搜索照片发现，龟峰景区的盐风化地貌更加普遍，而恰恰是这些洞穴对龟峰景区独特的地质景观起到了关键的装饰作用。为了弥补缺憾，我们于2021年10月自驾去了龟峰景区，在管委魏晨蕾老师带领下考察了其主要景点。本文就专门介绍该区的盐风化地貌，顺便探讨一下砂岩表面龟裂纹的成因</span><span style="font-family: 宋体; color: blue;">。</span></strong></span> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><strong><span style="font-family: 宋体; color: blue;"></span></strong></span></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=Slide7.JPG&id=826647" style referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=Slide9.JPG&id=826648" style referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=Slide8.JPG&id=826649" style referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=Slide15.JPG&id=826650" style referrerpolicy="no-referrer"></p><p><img src="https://cors.zfour.workers.dev/?http://bbs.sciencenet.cn/home.php?mod=attachment&filename=Slide11.JPG&id=826651" style referrerpolicy="no-referrer"></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><strong><span style="font-family: 宋体; color: blue;"></span></span></p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue; font-weight: bold;">    弋阳龟峰位于武夷山北麓与信江平原交界带，属于低山丘陵区。出露的主要地层为晚白垩世沉积形成的圭峰群（据Kusky et al, 2010</span><span style="font-size: 20px; color: blue; font-weight: bold;">；李馨敏等，</span><span style="font-size: 20px; color: blue; font-weight: bold;">2018</span><span style="font-size: 20px; color: blue; font-weight: bold;">），由巨厚层洪积砾岩和少量砂岩（及含砾砂岩）组成（见下图）。</span></span> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue; font-weight: bold;">    地层整体上向北西方向倾斜，倾角一般不超过20</span><span style="font-size: 20px; color: blue; font-weight: bold;">度。降水沿着</span><span style="font-size: 20px; color: blue; font-weight: bold;">NE</span><span style="font-size: 20px; color: blue; font-weight: bold;">和</span><span style="font-size: 20px; color: blue; font-weight: bold;">NW</span><span style="font-size: 20px; color: blue; font-weight: bold;">两组断裂切割，形成北东向和北西向交叉的一系列沟谷，将山体切割成一个个四边形柱体，而块体的东南边缘往往是陡峭的山崖，而其东北边缘和西南边缘也有陡崖发育，只是这些陡崖东段高而西段低而已。宏观上，龟峰景区被断裂切割的岩块多有着馒头状的浑圆顶部，加之局部有差异风化形成的突出岩石，这便形成了大小不等的龟状地形，也是龟峰景区的重要特征之一。显然，这些龟背形小山丘的形成与本区暴雨冲刷关系密切，年平均降水量近</span><span style="font-size: 20px; color: blue; font-weight: bold;">2000mm  </span><span style="font-size: 20px; color: blue; font-weight: bold;">就是重要的动力基础，而山脚下松散坡积物表面的流痕就是暴雨冲刷的证据。本篇博文不再详细讨论龟背地貌的形成过程，而是重点探讨一下崖壁上洞穴的成因。</span></span> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue; font-weight: bold;">    </span><span style="font-size: 20px; color: blue; font-weight: bold;">关于丹霞地貌区崖壁上洞穴成因，以前有很多猜测式判断，但都缺少充足的证据。我和苏德辰等（吕洪波等，</span><span style="font-size: 20px; color: blue; font-weight: bold;">2017</span><span style="font-size: 20px; color: blue; font-weight: bold;">）对丹霞山世界地质公园崖壁上的洞穴进行初步研究，明确指出其成因为盐风化作用，而且强调盐风化是物理风化作用。接着，我又在科学网上发布博文（吕洪波，</span><span style="font-size: 20px; color: blue; font-weight: bold;">2017</span><span style="font-size: 20px; color: blue; font-weight: bold;">），进一步以图文形式进行了详细解读。</span></span> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px; font-family: 宋体, SimSun;"><span style="font-size: 20px; color: blue; font-weight: bold;">    近年来，东华理工郭福生教授带领的团队对上述洞穴开展了广泛的野外考察调研，提出了“雨水冲刷”、“砾石脱落”的成因机制解（如：李馨敏等，2018</span><span style="font-size: 20px; color: blue; font-weight: bold;">），接着发表论文提倡多成因说（如：</span><span style="font-size: 20px; color: blue; font-weight: bold;">CHEN Liuqin et al., 2019</span><span style="font-size: 20px; color: blue; font-weight: bold;">）。</span></span> </p><p style="margin-top: 0;margin-bottom: 0;margin-left: 0in;direction: ltr;unicode-bidi: embed"><span style="font-size: 20px;"><span style="font-family: 宋体;
