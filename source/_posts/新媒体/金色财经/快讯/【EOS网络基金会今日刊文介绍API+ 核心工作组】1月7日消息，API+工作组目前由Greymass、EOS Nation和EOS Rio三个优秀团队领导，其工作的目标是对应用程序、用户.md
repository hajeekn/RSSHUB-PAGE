【EOS网络基金会今日刊文介绍API+ 核心工作组】1月7日消息，API+工作组目前由Greymass、EOS Nation和EOS Rio三个优秀团队领导，其工作的目标是对应用程序、用户和交易所的需求进行彻底调查，进一步了解开发者需求以便推出更棒的API服务，帮助开发者更简单的在EOS上创建项目的同时，推动更多EOS生态以外的项目更便捷的集成EOS网络。目前，API+、Core+、Wallet+ 和 Audit+ 四个核心工作组蓝皮书已接近完成，预计将于2022年1月下旬正式发布，EOS社区会审查和讨论其内容，来帮助确定在哪些方面优先分配资金并执行API+ 工作组提出的需求。
