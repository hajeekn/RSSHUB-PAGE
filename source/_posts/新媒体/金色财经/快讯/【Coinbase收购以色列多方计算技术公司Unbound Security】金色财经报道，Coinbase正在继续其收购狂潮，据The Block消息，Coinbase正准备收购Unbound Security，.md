【Coinbase收购以色列多方计算技术公司Unbound Security】金色财经报道，Coinbase正在继续其收购狂潮，据The Block消息，Coinbase正准备收购Unbound Security，这是一家专门从事多方计算 (MPC) 技术的以色列加密托管基础设施服务提供商。Coinbase 发言人表示，Coinbase 计划将 MPC 技术整合到其零售和机构产品中，目前这笔收购交易的具体金额尚未对外披露，迄今为止，Unbound Security 已筹集了至少 4000 万美元的外部资金，员工人数超过 70 人，预计全部收购交易会在几个月后完成。收购交易完成后，Coinbase还打算拓展以色列业务并将在该国一个“卓越技术中心”。 Unbound Security 团队将成为这个新研究设施的核心工作。
