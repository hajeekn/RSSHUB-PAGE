【NFT赛马游戏Zed Run发布2022年路线图】12月31日消息，NFT赛马游戏Zed Run发布2022年路线图，要点如下： 
1. 改进等级制度，时间为第一季度。 
2. ZED RUN代币。代币分发将分为三个主要类别：游戏内活动；增长激励；现有用户。 
ZED RUN代币将分四个阶段启动： 
第一阶段：初始空投（第一季度），将空投给符合条件的马厩主人和ZED RUN NFT持有者。分配将基于以下因素：拥有的赛马数量；拥有的赛马类型；ZED RUN NFT所有权的时间。 
第二阶段：赛马比赛（第一季度）。 
第三阶段：赛马退役（第二季度）。 
第四阶段：租赁（第二季度）。 
3. 终极锦标赛和赛马体验（从第一季度开始）。
