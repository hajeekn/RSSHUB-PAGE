<p>國慶連假期間，區塊勢正式成為 LikeCoin 的區塊鏈節點（node），或稱驗證者（validator）。如果你已經熟悉 LikeCoin 的治理機制或是手上有 LIKE，歡迎<a href="https://stake.like.co/validators/cosmosvaloper18u8zdrgngyfeq2rfh9ds7a3hg7awx8h9kpw2xp" target="_blank">委託</a>給區塊勢，可以替你賺得每年 9.89% 的收益（浮動）並參與鏈上治理。</p><p>這篇文章介紹 LikeCoin 驗證者在做什麼、有哪些成本與回報，以及你可以如何參與？</p><h2>鏈上立委</h2><p>LikeCoin 驗證者就是 LikeCoin 區塊鏈的礦工。</p><p>多數的加密貨幣都是建立在以太坊區塊鏈上，只要以太坊安全無虞，這些加密貨幣的「地基」就很牢固。而 LikeCoin 則是在 Cosmos 生態系中獨立的區塊鏈，相當於自己打地基。因此，它得招募自己的礦工來維持 LikeCoin 區塊鏈營運。</p><p>為什麼要這麼大費周章？有經濟、政治兩大原因。</p><p>LikeCoin 是去中心化出版（DePub）的基礎建設。人們每次按下讚賞鍵，相當於發送一筆微型支付（LIKE 幣）給文章創作者。若按照目前以太坊的<a href="https://etherscan.io/gastracker" target="_blank">擁堵狀況</a>，每筆轉帳的手續費肯定遠高於打賞金額。</p><figure class="image"><img src="https://assets.matters.news/embed/fb4e2def-a6b2-42cb-9852-0872886b650e.png" data-asset-id="fb4e2def-a6b2-42cb-9852-0872886b650e" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>誰會為了打賞 5 元台幣，而支付 600 元的轉帳手續費？這就是 LikeCoin 獨立建區塊鏈的經濟理由。</p><p>其次，以太坊上的應用開發者眾多。優點是生態系豐富，讓不同應用的功能就像是疊樂高一樣，可以相互整合；缺點則是底層的區塊鏈無法為單一應用最佳化。</p><p>以 Compound 為例，它的去中心化治理僅限於 Compound 的借貸協議本身。若新功能需要改動以太坊的原有功能，那就得到以太坊的治理論壇另外提案。由另外一群人經過長時間討論，確保新功能符合各方利益後，才有可能實現。</p><p>相對之下，LikeCoin 獨立建區塊鏈，可以自由調整所有設定。小到功能升級，大至經濟模型都可以提案修改。而 LikeCoin 驗證者的角色就像是「鏈上銀行」加上「鏈上立委」，LIKE 的持有者則是「鏈上公民」。</p><p>人們若將 LIKE 委託給驗證者，不只可以替你賺取收益，也等於是在參與鏈上治理。但驗證者並不是無償的志工，他們得替自己「開薪水」，並說服人們將手上的 LIKE 委託給他。</p><h2>成本與回報</h2><p>如下圖所示，目前 LikeCoin 鏈上共有 50 位驗證者。如果你手上有一筆 LIKE，你會選擇抵押給誰？</p><figure class="image"><img src="https://assets.matters.news/embed/1786235a-1400-4677-8e47-46ef05601bac.png" data-asset-id="1786235a-1400-4677-8e47-46ef05601bac" referrerpolicy="no-referrer"><figcaption><span></span></figcaption></figure><p>就像拿到選舉公報時，人們會先找熟悉的名字。只不過，畫面右邊還有兩個供人們參考的關鍵數據 —— 收益率（Rewards）以及決策權（Voting Power）。</p><p>收益率表示你將 LIKE 抵押給這位驗證者，每年可以獲得的收益回報。以畫面最上方的兩位驗證者 WeLike、0-ZeroKeeper 為例，委託給他們的收益率高達 10.99%，是目前鏈上收益最高的驗證者。這個數字怎麼來的？</p><p>它與 LikeCoin 的年增發率（inflation rate）、參與抵押的比例高低（delegation ratio），以及驗證者收取的佣金（commission）有關係。</p><p>舉例來說，目前 LikeCoin 的年增發率是 7%。也就是 LIKE 在明年的市場流通總量，理論上會比今年增加 7%。但增發的這 7% 並非人人有獎。只有參與抵押的人，才可以獲得這些新發行的 LIKE。</p><p>假設市場上 100% 的 LIKE 都參與抵押，而驗證者收取的佣金為 0％，那麼年收益率就會是 7%。但如果只有 50% 的 LIKE 參與抵押，驗證者同樣將佣金設為 0％，收益率就會增加到 14%。</p><p>簡單來說，如果你沒把 LIKE 抵押給驗證者，就等於放棄領取每年增發的 LIKE。雖然你手上的 LIKE 數量不變，但在市場流通量的佔比則會被稀釋掉。</p><p>而驗證者設定的佣金，就像是自己開出來的「薪水」。代表他們會從抵押者每年多獲得的 LIKE 中，收取多少比例當成驗證者的回饋獎勵。</p><p>像是 WeLike、0-ZeroKeeper 這兩位驗證者，之所以能夠提供高達 10.99% 的年收益率，就是因為他們自願當志工不領薪水。反觀列表上收益率為 5.49％ 的驗證者，就是開出 50% 的佣金率。</p><p>人們如果不清楚驗證者實際上會做出哪些貢獻，當然會優先抵押給不收佣金的志工。而收取高額佣金的驗證者可能正在貢獻社群，但也可能就只是「坐領乾薪」。和民意代表的狀況類似。</p><p>以驗證者 <a href="https://stake.like.co/validators/cosmosvaloper1v8njts96gl5eqstnen4gksdy5k860fau65c3sw" target="_blank">Forbole</a> 為例，他們開出的佣金率是 100%。換句話說，你抵押的 LIKE 收益都會直接進到 Forbole 的口袋裡，相當於捐贈未來的 LIKE 收益給他們。不過 Forbole 對 LikeCoin 的貢獻也非常具體，就是為 LikeCoin 打造類似 Etherscan 的區塊鏈瀏覽器 —— <a href="https://likecoin.bigdipper.live/" target="_blank">Big Dipper</a>。這是區塊鏈的基礎建設之一，否則其他用戶就得自己用命令行撈出鏈上數據。</p><p>從這個角度來看，將 LIKE 委託給有收佣金的驗證者，相當於認同他們在做的事，並直接以未來 LIKE 幣的部分收益贊助他們。</p><p>這比定期定額捐款更簡單。如果未來幣價上漲，所有人都能雨露均霑。反過來說，驗證者也得定期向委託人交代自己做了哪些事、對於社群治理的想法，才能收到更多的 LIKE 委託。</p><p>以區塊勢為例，我暫訂 10% 佣金是用來支付節點的營運成本，並會在未來每次投票前寫文章說明支持或反對的理由。</p><h2>流動民主</h2><p>有別於大家熟悉的選舉制度，每隔幾年就會改選，鏈上立委沒有任期限制。</p><p>只要驗證者收到的 LIKE 抵押數量在前 50 名，就能即刻「當選」。反過來說，人們要「罷免」鏈上立委只要將 LIKE 改抵押給其它驗證者，讓他跌出前 50 名就算是罷免成功。</p><p>鏈上立委可以說是 24 小時全年無休都在競選狀態。他們必須理解 LikeCoin 用戶的需求，提出政見並爭取支持。支持度越高的驗證者不只決策權越大，能獲得的收益理論上也就越可觀。這是一場賽局。驗證者必須權衡各項因素，才能獲得最高支持。</p><p>只不過，目前 LikeCoin 的治理生態還沒有這麼成熟。多數人還是以收益率為主要考量，而非去細究每位驗證者的貢獻與佣金的關係。就像 Compound 的治理論壇中，會直接呈現每位民意代表參與過多少提案以及支持（或反對）哪些提案，讓人們當成委託的參考依據。</p><p>未來 LikeCoin 肯定也會有類似的治理數據，讓大家在支付「薪水」、委託決策權之前能對每位驗證者有大致的理解。如果你也希望未來的每次投票提案，都可以有一篇像這樣的文章交待來龍去脈，那歡迎你下載 <a href="https://liker.land/getapp" target="_blank">Liker Land</a> 這款錢包 app 並將手上的 LIKE 委託給區塊勢。用幣投票之餘，還能一起體驗全新的<a href="https://ckxpress.com/liquid-democracy-with-audrey-tang/" target="_blank">流動民主</a>。</p><hr><p>如果你喜歡這篇文章，說不定你也會對區塊勢的<a href="https://blocktrend.pse.is/archive"
