noreferrer" target="_blank">公佈讚賞公民Web3 的要求及委託節點有獎</a>，更開展了兩個新議案：</p><h2>議案 36</h2><p><a href="https://stake.like.co/proposals/36" rel="noopener noreferrer" target="_blank"><strong>指引式議案：賦權 @ckxpress 成立 Tech Subdao</strong></a></p><p>這是個指引式議案 Signaling Proposal，如果不知道是甚麼的話請參看 <a href="https://matters.news/@daisy/like-coin-%E8%AD%B0%E6%A1%88%E7%9A%84%E6%8F%90%E6%A1%88-%E6%8A%95%E7%A5%A8%E8%88%87%E5%9F%B7%E8%A1%8C-bafyreifjwgqayl6o65ej4wikyglmnlnflt463s7ahef5blm7polpqa77ke" target="_blank">ckxpress</a> 成立一個 3 至不超過 5 人的 Tech Subdao，即是在 LikeCoin DAO 以下的小 DAO。這個 Subdao 將負責：</p><ol><li>識別 LikeCoin 的技術需求，包括協議、區塊鏈、工具及 DApps</li><li>尋找願意在 LikeCoin 上構建的開發者、供應商及企業家</li><li>管理與 2. 這班人的關係，並公平及時地向他們付款</li><li>每月向社群匯報</li></ol><p>由於這個議案只是想取得太家的意向，所以議案本身並不會進行任何請款行為。如有需要將會獨立向社群提出社群池支出議案 Community Pool Spend Proposal，就好像 <a href="https://matters.news/@daisy/like-coin-%E8%AD%B0%E6%A1%88-33-%E4%BB%A5%E7%A4%BE%E7%BE%A4%E6%B1%A0%E5%83%B1%E7%94%A8-oursky-%E9%96%8B%E7%99%BC%E4%BA%BA%E5%93%A1-bafyreialwc6kn5lvtg3lxcdkpcjv4f2emmju5mpdyr3umxahyeyqeyjkde" rel="noopener noreferrer" target="_blank">LikeCoin 議案 33：以社群池僱用 Oursky 開發人員</a> 的做法，向社群請款。</p><p>如希望參與討論，可到這條 <a href="https://discord.com/channels/763001015712350231/943338228805677066/943338231661985823" rel="noopener noreferrer" target="_blank">Discord 討論串</a>。</p><h2>議案 37</h2><p><a href="https://stake.like.co/proposals/37" rel="noopener noreferrer" target="_blank"><strong>社群池支出議案：Osmosis 外部激勵</strong></a></p><p>正如 <a href="https://matters.news/@daisy/%E8%B3%BC%E8%B2%B7-like-coin-%E5%83%B9%E6%A0%BC%E8%B3%87%E8%A8%8A-%E7%9B%A4%E9%BB%9E%E5%A0%B4%E5%A4%96%E4%BA%A4%E6%98%93%E5%95%86-%E7%82%BA%E8%AE%9A%E8%B3%9E%E5%85%AC%E6%B0%91web3-%E5%81%9A%E6%BA%96%E5%82%99-bafyreig6p7dfzvb2v74qcb2fk3b6dxyxkgmjzycnomxaizsmcxmmjxx3b4" rel="noopener noreferrer" target="_blank">購買 LikeCoin、價格資訊、盤點場外交易商，為讚賞公民Web3 做準備</a> 一文所述。成立 Osmosis 外部激勵的議案終於出現了！</p><p>議案內容主要是為 Osmosis 上的 ATOM/LIKE 和 LIKE/OSMO 池即 553 和 555 池請款去配對外部激勵 15,000,000 LIKE，以吸引新的 LikeCoin 持有者加入提供流動性。</p><p>如果議案獲得通過，LIKE 將會被傳送到 Liker Land 擁有的錢包地址 cosmos1fr62zx24gapdul0gze8lt3y9fmtt7h6mjv5w6c，這個地址將會在議案通過後將資金分配到上述兩個池中，以配對 Osmosis 的內部激勵。</p><p>這個外部激勵計劃將持續 90 天，大致為：</p><p>ATOM/LIKE = 83,333.33 LIKE/天</p><p>LIKE/OSMO = 83,333.33 LIKE /天</p><p>如希望參與討論，可到這條 <a href="https://discord.com/channels/763001015712350231/933303531648200755" rel="noopener noreferrer" target="_blank">Discord 討論串</a>。</p><hr><p>議案 36 某程度上是承接上年完結了的 LikeCoin Grants 接下來的事情。依照 <a href="https://blog.like.co/likecoin-community-update-202202-17c025a79820" rel="noopener noreferrer" target="_blank">本月 Community Call 匯報</a>，如果這個 Tech Subdao 成立以後將專注於技術發展。而以往 LikeCoin Grants 包含的推廣及內容保存範籌就會由另外成立的 Subdao 去處理。由我偶像領軍的 Subdao 我當然會亳不客觀地支持吧。</p><p>至於議案 37 分明是給領完空投的大家一些好康：甚麼也不做就能從池子收到 LIKE 與 OSMO 雙重獎勵，實在太讚！而吸引不認識 LikeCoin 項目的人去了解更多，亦是非常重要。</p><p>故此毫無懸念，兩個議案我都會投 YES！</p>
