<p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;"><strong>预配的模块支持通过 Wi－Fi 和蜂窝<span class="hrefStyle"><a href="https://tele.ofweek.com/" target="_blank" title="通信">通信</a></span>物联网与 AWS 进行开箱即用的安全通信</strong></p><center><p style="text-align:center"><img src="https://cors.zfour.workers.dev/?http://images.ofweek.com/Upload/News/2022-01/06/yuanchen/1641463050148014072.jpg" alt="u-blox IoT方案支持与AWS云安全通信" title="u-blox IoT方案支持与AWS云安全通信" referrerpolicy="no-referrer"></p></center><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">作为一家提供定位和无线<span class="hrefStyle"><a href="https://tele.ofweek.com/CATList-8320500-8300-tele.html" target="_blank" title="通信技术">通信技术</a></span>及服务的全球领先供应商，u－blox （SIX：UBXN） 宣布推出NORA－W2 AWS IoT ExpressLink Wi－Fi 模块和用于物联网 （IoT） 通信的 SARA－R5 AWS IoT ExpressLink 蜂窝通信模块来帮助设备和车队管理实现开箱即用的 Amazon Web Services （AWS） 云服务，AWS IoT ExpressLink 模块将显著加快设计和上市时间。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">对许多企业来说，将设备连接到云端仍然充满挑战。目前市面上的许多解决方案需要安装和维护专用软件来管理设备通信。此外，许多传统设备资源过于紧张，无法容纳实现云通信所需的额外代码。为了应对这些挑战，EDN， Inc． 为其园艺产品选择了这两个模块：</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">“我们当时希望找到一种快速有效的方法，来为我们的室内种植平台实现强大的云功能。根据既往经验，我们知道这样做会增加时间、资金和整体产品上市计划方面的重大风险，”EDN， Inc． 首席执行官 Ryan Woltz 表示。随后他又说道，“物联网设备的通信技术十分复杂，因此我们的团队只能将开发外包给收费高昂的第三方，或者安排内部的工程资源，而这两种方案都可能给延迟产品创新功能的开发，拖慢上市速度。但现在借助 u－blox 的 NORA－W2 和 SARA－R5 AWS IoT ExpressLink 模块，我们可以在短短几天内完成从原型设计到生产的过程，并建立可扩展、安全可靠的云通信。如今，u－blox 平台已成为我们的战略优势，使我们能专注于自身最擅长的领域 — 打造创新产品，让人们可在室内轻松乐享自然美景。”</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">u－blox AWS IoT ExpressLink 模块均致力于优化速度和简单性。这些模块不需要在设备端运行和维护复杂的软件，为快速设计铺平了道路，是面向消费者市场和工业市场的中小型企业的理想之选。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">u－blox AWS IoT ExpressLink 模块具备简化且高度抽象化的命令集，不需要具备物联网产品开发、网络和加密方面的专业能力即可轻松进行配置。它们预配为与 AWS IoT Core 进行通信，只需要运行几个命令即可开箱即用地安全连接到 AWS，还可以与 AWS 生态系统中的服务集成。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;"><strong>支持 Wi－Fi 和蜂窝通信</strong></p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">这两款模块填补了 Wi－Fi 和蜂窝 AWS IoT ExpressLink 模块的市场空白。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">NORA－W2 AWS ExpressLink 模块是 u－blox 全新推出的一款模块，支持 Wi－Fi 4 通信以及蓝牙5低功耗通信（包括蓝牙远距离通信）。这一模块结构紧凑（10．4 x 14．3 毫米）并且搭载强劲的双核中央处理器 （CPU），配备天线引脚或印刷电路板 （PCB） 天线，适用于高端应用。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;"><strong>SARA－R510 AWS IoT ExpressLink 模块是 SARA－R510S LTE－M 和 NB－IoT 模块的一种变体。低功耗广域 （LPWA） 模块经过功耗优化，在省电模式 （PSM） 下消耗的电流小于 1 </strong>μA，是电池供电型应用的理想之选。该模块已取得多区域使用认证。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;"><strong>为刚刚涉足物联网领域的企业提供便利</strong></p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">u－blox 产品中心执行总监兼联合创始人 Andreas Thiel 表示：“借助 NORA－W2 和 SARA－R5 IoT ExpressLink 模块，无线设备可以快速连接到云。AWS IoT ExpressLink 能简化对云服务的访问，从而消除了新进入物联网领域的企业面临的关键阻碍之一：复杂性。”</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">AWS 消费者 IoT 与通信部总经理 Shyam Krishnamoorthy 表示：“对 AWS 来说，与 u－blox 合作是一项明智的决策，因为我们知道，他们面向工业和专业应用的高质量模块将为我们的各种物联网相关软件和云服务提供有力补充。NORA－W2 和 SARA－R5 模块涵盖 Wi－Fi 和 LTE－M，可以满足物联网通信的关键需求，并帮助客户缩短上市时间”。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">u－blox AWS IoT ExpressLink 模块和 EVK 的首批样品将于 2022 年第一季度上市。</p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">u<strong>blox 简介</strong></p><p style="margin-top: 10px; margin-bottom: 10px; white-space: normal; text-indent: 2em; text-align: left;">瑞士 ublox 公司 （SIX：UBXN） 是为汽车、工业和消费市场提供定位和<span class="hrefStyle"><a href="https://tele.ofweek.com/CAT-8320501-WX.html" target="_blank" title="无线通信">无线通信</a></span>技术的领先供应商。其解决方案能准确进行人员、车辆和机器的定位，并通过蜂窝网络和短程通信网络实现无线通信。凭借广泛及多样化的芯片、模块和日益扩大的产品支持数据服务生态系统，ublox 公司在帮助其客户针对物联网快速而成本高效地开发创新解决方案方面具有得天独厚的优势。u－blox 公司总部设在瑞士塔尔维尔，目前在欧洲、亚洲和美国都设有办事处。 您可以在Facebook， LinkedIn， Twitter ＠ublox 和YouTube找到我们。</p>
