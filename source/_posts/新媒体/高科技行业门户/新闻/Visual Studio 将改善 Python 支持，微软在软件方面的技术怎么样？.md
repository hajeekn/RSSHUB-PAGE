<p style="text-indent: 2em; text-align: left;">近日，随着 Visual Studio 2022 的发布，微软正在更改支持从 Visual Studio 安装程序安装 Python 运行时的方式。</p><p style="text-align:center"><img src="https://mp.ofweek.com/Upload/News/Img/member48043/202112/13231417280619.jpg" alt="Visual Studio 将改善 Python 支持，微软在软件方面的技术如何？" title="Visual Studio 将改善 Python 支持，微软在软件方面的技术如何？" referrerpolicy="no-referrer"></p><p style="text-indent: 2em; text-align: left;">根据微软官博的介绍，Visual Studio 的生命周期和 Python 的生命周期不一致，这就导致 Visual Studio 有时捆绑了旧版的 Python。</p><p style="text-indent: 2em; text-align: left;">比如，Visual Studio 2019 的目标是 Python 3．7，而 Visual Studio 2022 的目标是 Python 3．9。每年 Python 都会发布新版本，并且二进制安装程序在发布后的 1．5 年内可用于 Windows。这意味着 Python 3．9 的最后一个二进制安装程序更新将在 2022 年 3 月左右发布，之后将不再获得支持。微软对于 Visual Studio 旧版本的做法是，将该 Python 版本标记为“不受支持”，但不会删除这些版本，也不会添加更新的版本。</p><p style="text-indent: 2em; text-align: left;">智慧芽专家表示，截至最新，微软及其关联公司在126个国家／地区中，共有16万余件专利申请。从专利申请人上来看，该公司目前的专利申请人排名中分别来自微软技术许可有限责任公司、微软公司、领英。</p><p style="text-indent: 2em; text-align: left;">此外，截至最新，微软及其关联公司在126个国家／地区中，共有8820件软件直接相关专利申请。从专利创新词云可知，近几年的专利布局主要集中于应用程序、软件应用、计算机、用户界面、计算设备等专业技术领域。</p><p style="text-indent: 2em; text-align: left;">（备注：智慧芽全球专利数据库收录数据包括126个国家／地区中已经公开的专利，一般来说，专利从申请到公开可查询，需要4到18个月）</p>
